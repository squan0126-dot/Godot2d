# 序章小说式描述

> 设计意图：以"烟笼寒水月笼沙，夜泊秦淮近酒家。商女不知亡国恨，隔江犹唱后庭花"为意境蓝本，用场景本身呈现乱世中的虚假繁华。首尾呼应终章"盛唐幻境"。

---

## 开场：秦淮

---

水声。

李长安睁开眼，入目是一片月色。

他躺在什么东西上。晃的。身下是木板，耳边是水拍船舷的声响。

夜风裹着脂粉香拂过面颊。

他偏过头。

一只白皙的手正递到嘴边，指尖捏着一颗紫莹莹的葡萄，饱满欲裂，汁水沿指缝淌下来。

"公子醒了？"

声音娇软，带着笑意。

李长安顺着那只手往上看——一张桃花面，柳眉杏目，鬓边簪着一朵芍药，正含笑望着他。

她身后是绢纱帐幔，随风轻摆。帐外透进来的光是暖的，灯笼映在水面上，碎成一河金鳞。

船头传来琵琶声。

李长安撑起身子。

这是一艘画舫。

不小。前舱宽敞，铺着锦毯，摆着矮案，案上是瓜果酒盏。后舱帘幕低垂，隐约有人影绰绰。

船头坐着两个女子，一个抱琵琶，一个执檀板。

唱的是——

*玉树后庭花，花开不复久。*

嗓音清婉，和着水声，悠悠荡荡散在夜色里。

河面上不止这一艘船。远远近近，十数盏灯笼浮在水上，有笑声从别的船上传来，有人在划拳，有人在唤酒。

两岸是酒楼茶肆，灯火通明。

太平。

富贵。

像是盛世该有的模样。

那女子又递了一颗葡萄过来。

"公子方才睡得沉，奴家唤了好几声呢。"

李长安下意识张了嘴。

葡萄入口。凉的。甜的。

他咽下去。

然后——

胸口一沉。

像是有什么东西突然压上来。不重，但冰凉，贴着皮肉。

李长安低头。

衣襟里不知何时多了一本书。

黄壳。薄薄的。封皮上没有字。

他抽出来，翻开。

首页空白。

不对——有字在浮现。像是有人用看不见的笔，一笔一划地写。

墨迹洇开。

两个字。

**"通幽。"**

---

## 幻象崩塌

---

李长安的瞳孔骤缩。

不是因为这两个字。

是因为——

他看见了。

方才递葡萄的那只手，还停在半空。

白皙。纤细。

但指甲——

指甲底下是黑的。不是污垢。是腐肉。指尖的皮肤在往下坠，像手套一样松垮地挂着，露出底下灰白的骨节。

李长安猛地抬头。

那张桃花面还在笑。

但她的左眼——

左眼眶是空的。

一截蛆从眼眶里探出头，蠕了蠕，又缩回去。

她还在笑。嘴角的肉已经烂穿了，露出里面的牙根和颌骨。芍药花簪在头上，底下的头皮大片大片地脱落，露出青灰色的颅骨。

"公子？"

她又递了一颗过来。

李长安的目光落在她指尖。

不是葡萄。

是一颗眼珠子。

紫红色的虹膜，还带着血丝，被她两根白骨指头捏着，汁水顺着指缝往下淌。

船头的琵琶声还在响。

李长安转头看过去。

抱琵琶的女子——没有下半身。

她的腰以下是空的，脊椎骨直接插在船板上，内脏拖在地上，被风吹得轻轻晃荡。她的手指还在拨弦，十根指骨没有皮肉，骨头敲在弦上，发出干涩的"铮铮"声。

执檀板的那个更不成人形。

一张人皮，被撑在木架上。风灌进去，鼓起来，瘪下去，一鼓一瘪之间，檀板就被"拍"响一下。

*玉树后庭花。*

*花开不复久。*

还在唱。

声音从那张人皮的嘴洞里挤出来，像风穿过破瓮。

李长安猛地站起来。

船晃了一下。

他扶住船舷，往河面上看——

那些灯笼还浮在水上。

但底下不是船。

是尸体。

一具具泡得发胀的尸体仰面漂在河上，肚皮鼓起来，白花花的，灯笼就插在它们的肚脐眼里。烛火从腹腔里透出来，把肚皮照得通红，像一盏盏人肉灯笼。

两岸的酒楼——

没有酒楼。

是烧焦的断壁残垣。焦黑的房梁戳在夜空里，像一排排肋骨。有的墙上还挂着半截尸体，烧得蜷缩成一团，分不清男女。

河水是红的。

不是灯笼映的。

是血。

浓稠的，泛着腥气的血水，裹着碎肉和断指，缓缓地流。

李长安低头看脚下。

没有画舫。

几块破木板用烂绳子绑在一起，勉强浮在水面上。木板上有刀痕、有烧痕、有干涸的血迹——这是战船的残骸，被河水冲散后，碎片拼成了一方筏子。

锦毯没有。矮案没有。瓜果酒盏没有。

只有他，站在一堆破木板上，在一条血河里飘着。

那个桃花面的女子——

一具骷髅蹲在筏子边沿，空洞的眼眶对着他，颌骨一张一合。

"公子，再吃一颗嘛。"

李长安后退一步。脚下的木板"嘎吱"一声，裂了条缝。

血水从缝里涌上来，漫过他的脚面。

---

## 老道士

---

远处，传来一声苍老的呵斥——

"孽畜！"

一道黄光从岸边激射而来，"啪"地打在那骷髅身上。骷髅尖叫一声，散了架，骨头哗啦啦落进水里。

船头那张人皮也被黄光扫中，"嗤"地一声烧起来，蜷缩，焦黑，化作灰烬。

琵琶声断了。

筏子剧烈摇晃。李长安单膝跪下，死死抓住木板边缘。

一个人影站在岸边。

月光下看不清面目，只见一身灰旧道袍，手里捏着张黄纸。

"后生。"

那人的声音沙哑，像石头磨石头。

"水里不干净。上来。"

---

李长安踩着没膝的血水蹚上岸。

回头看那条河。

月色底下，河面平静，浮尸缓缓顺流而下。远处的火光还没灭尽，映着半边天，像是一整座城在烧。

风把灰烬吹过来，落在他肩上。

那老道士站在他身后，也在看那条河。

沉默了一会儿。

"方才那些东西，"老道士开口，"不算厉害的。战死的人太多，怨气聚在一处，成了气候。它们生前是秦淮河上的船妓，死了也不肯走，还在河上拉客。"

他顿了顿。

"拉活人的魂魄，喂自己。"

李长安低头看了看手里的黄壳书。

"通幽"两个字还在首页上，墨迹未干。

"你能看见它们，"老道士瞥了一眼那本书，目光微动，"是因为这个。"

他没多解释。转过身，往岸上走。

"跟上。天亮前得离开河边。夜里的东西，比方才那几个饿殍厉害得多。"

李长安攥紧书，跟了上去。

身后，河面上最后一盏"灯笼"沉入水底。

尸体翻了个身，沉下去了。

河面恢复了死寂。

只有风，还在吹。

## 设计备注

| 要素 | 处理方式 |
|------|----------|
| **"商女不知亡国恨"意境** | 船妓死后仍在河上卖笑拉客——她们"不知"自己已死，"不知"城已破国已亡，还在唱《玉树后庭花》。用场景呈现，不引用原诗 |
| **"烟笼寒水月笼沙"** | 月色、河面、灯笼倒影——开场的视觉基调就是秦淮夜泊的画面感 |
| **"夜泊秦淮近酒家"** | 画舫、两岸灯火酒楼——幻象中的繁华就是"近酒家"的感觉 |
| **"隔江犹唱后庭花"** | 城在烧、人在死、河里全是尸体——琵琶声没停，还在唱。隔着一条血河，亡国曲犹在 |
| **通幽能力** | 黄壳书出现→解锁通幽→立刻看见真相。因果清晰，不需要额外解释 |
| **老道士救场** | 一声呵斥+黄符驱鬼，干脆利落。不废话，先救人再说 |
| **战争废墟** | 不是"鬼造的幻象"，而是"死人的执念"——战争毁了一切，但死去的船妓还在重复生前的事。更悲凉 |
| **筏子真相** | 战船残骸碎片拼成的木板筏——直接交代"这里刚经历过水战" |
| **与终章呼应** | 序章是"一条血河上的假画舫"，终章是"一座废城伪装成盛唐"——同一母题的微缩版与完整版 |
| **恐怖递进** | 繁华画舫→胸口异物→指甲腐肉→空眼眶→眼珠葡萄→半身琵琶女→人皮檀板→尸体灯笼→血河废墟→骷髅索命 |
