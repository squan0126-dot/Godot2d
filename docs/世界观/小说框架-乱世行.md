# 《乱世行》小说框架

> **类型**：穿越 · 侠义 · 志怪 · 家国
> **文风参考**：《地煞七十二变》——轻快江湖气，底色是家国情怀
> **故事时间线**：907年（后梁建立）→ 927年（赵匡胤出生）
> **核心结局**：主角护送英雄亡魂投胎，最后发现那户人家——姓赵

---

## 一、核心立意

```
一个现代人穿越到五代乱世，
他没有改变历史的野心，
只是见到了太多不该死的人，
见到了太多死了还放不下的魂，
见到了一个本该并肩走完这段乱世的英雄，
被人害死，连转世都被人阻断。

他只想做一件事：
把这个英雄的魂，送到他该去的地方。

——他不知道，那个地方，
   将会改变整个天下。
```

---

## 二、主角设定

### 【主角：李归舟（现代人）】

```
【现代身份】
  - 姓名：李归舟（穿越后沿用，现代父母起的名字）
  - 职业：历史系研究生，专攻五代十国史
  - 性格：嘴贫、理性、有点丧，但关键时刻有底线
  - 特长：熟知这段历史的大致走向（但细节不一定准）

【名字的寓意——藏在名字里，不在故事里点破】
  - 李：唐朝国姓。故事从唐朝灭亡开始，到赵宋种下根基结束。
        李→赵，两朝更替，主角的姓氏本身就是这段历史的缩影。
        不需要在故事里反复强调，读者读完自然明白。
  - 归舟：谐音"归州"。燕云十六州，"归"是这个时代最深的遗憾，
          也是主角名字里藏着的、一生未必能亲眼看到实现的愿望。
          郭长风投胎后三十三年，赵匡胤才能黄袍加身；
          燕云十六州，要等到更久以后。
          "归舟"，是一个还没靠岸的名字。

【穿越方式】
  - 在图书馆研读《酉阳杂俎》时，
    被书中夹着的一张古符文吸入
  - 落点：907年，汴州城外，汴河边
  - 穿越时随身带了什么：
    一本《酉阳杂俎》（原书，成为金手指的载体）
    一部手机（没信号，但有离线地图和历史资料）
    一件现代外套（很快换掉）

【穿越后的处境】
  - 语言：古代汉语与现代普通话差异不算太大，
    加上历史系背景，勉强能交流
  - 身份：自报"李氏遗民，从关中逃来"
    （这个身份在乱世中既危险又好用）
  - 初始状态：身无分文，不会武功，
    但有现代人的逻辑思维和历史知识

【现代人价值观的体现】
  - 不认为"妖怪就该被消灭"——它们曾经是人
  - 不认为"皇帝天生高贵"——他只看这个人做了什么
  - 不认为"女人就该在家"——对崔晚烟等女性配角平等相待
  - 不认为"牺牲是理所当然的"——每一条命都值得被珍惜
  - 最大的现代特质：他会问"为什么"，而不是接受"本来如此"
  - 最大的现代特质：他会问"为什么"，而不是接受"本来如此"
```

## 三、金手指设定——《酉阳杂俎》残卷

```
【金手指名称】：《酉阳杂俎》残卷（以下简称"残卷"）

【外观】
  - 一本破旧的古籍，封面已模糊
  - 书页泛黄，但字迹清晰
  - 某些页面在特定条件下会自动翻动，显示新内容

【核心能力】
  ① 【志怪感知】
     - 主角能看到普通人看不到的鬼魂和妖怪
     - 能感知妖怪的"执念"——知道它们为什么在这里
     - 能判断妖怪的大致强弱（但不精确）

  ② 【残卷查阅】
     - 遇到特定妖怪时，残卷会自动显示该妖怪的信息：
       外形、习性、弱点、化解方式
     - 但信息是古文，需要主角自己翻译理解
     - 有时信息不完整，需要主角自己补全

  ③ 【执念化解】
     - 主角可以通过"理解执念"来化解鬼魂
     - 不是用法术强行驱除，而是找到执念的根源，帮助解决
     - 这是主角最核心的能力，也是他的处事方式

  ④ 【符文临摹】
     - 残卷中有部分符文，主角可以临摹使用
     - 但每次使用有代价（消耗体力/精力，严重时伤身）
     - 符文效果：驱散、封印、短暂保护——不能直接杀死妖怪

【金手指的限制】
  - 残卷不能预知未来（主角的历史知识才是"预知"）
  - 残卷对"人祸"无效——它只管妖怪，不管人心
  - 残卷会"成长"——随着主角化解的执念越多，
    残卷显示的内容越来越详细，能力越来越强
  - 残卷有意志——某些时候它会"拒绝"显示信息，
    主角需要自己想清楚才行

【金手指的隐藏设定（结局揭晓）】
  - 残卷本身就是那位英雄亡魂留下的东西
  - 英雄生前是一位道士/术士，将毕生所学封入残卷
  - 主角被残卷选中，是因为他"能理解这个时代的人"
  - 这也是为什么主角能帮助英雄亡魂完成最后的心愿
```

## 四、英雄亡魂——郭长风

```
【姓名】郭长风（字：破虏）
【生前身份】
  - 后梁初年的一位将领/游侠（非历史人物，架空）
  - 出身寒微，凭战功升至中层武将
  - 有统一天下、终结乱世的远大抱负
  - 同时精通道术，是残卷的原主人

【死亡经过】
  - 907年，后梁建立后，郭长风因反对朱温的暴政，
    被政敌陷害，以"通敌"罪名处死
  - 死时年仅三十二岁，抱负未竟，执念极深
  - 死后无法转世——政敌买通了阴间的"鬼差"，
    用邪术将其魂魄困在阳间，永世不得投胎
  - 这是一种极残忍的报复：
    不让你死，也不让你活，就让你永远飘荡

【郭长风的性格】
  - 豪迈、直率、有点莽，但心思细腻
  - 对天下苍生有真实的悲悯，不是口号
  - 死后仍保持生前性格，但多了一份沧桑
  - 与主角的关系：
    最初互相看不顺眼（一个太理性，一个太冲动）
    后来成为真正的知己——"并肩作战的兄弟"

【郭长风的执念】
  - 表层执念：我死得不明不白，要申冤
  - 深层执念：我的抱负没有完成，天下还没太平
  - 最深的执念：我相信这个天下会好的，
                 我只是没能亲眼看到

【郭长风为什么不能转世】
  - 政敌用"锁魂钉"将其魂魄钉在阳间
  - 锁魂钉分散在五个地方，需要逐一找到并拔除
  - 每拔一根，郭长风的魂魄就更自由一些
  - 全部拔除后，他才能真正投胎
  - ——这就是主线任务的结构

【结局的秘密】
  - 郭长风投胎的那户人家，姓赵
  - 郭长风转世后，就是赵匡胤
  - 赵匡胤身上那股"终结乱世、善待百姓"的气质，
    正是郭长风未竟抱负的延续
  - 主角不知道这一点，直到最后才发现
```

## 五、故事总体结构（三卷本）

```
第一卷：汴州乱局（907—912年）   ← 穿越落地，认识郭长风，明白任务
第二卷：五钉之路（912—922年）   ← 寻找五根锁魂钉，历经各地见闻
第三卷：归处（922—927年）       ← 最后一钉，护送投胎，揭晓身份
```

## 六、三卷详细框架

---

### 📖 第一卷：汴州乱局（907—912年）

**主题**：我穿越到了最乱的时代，然后遇到了一个死人

**开篇场景**：
```
907年秋，汴州城外，汴河边。

李归舟从河里爬出来，手里还攥着那本《酉阳杂俎》。
他环顾四周：土路、木桥、破旧的城墙、
穿着麻布衣服的流民……

他在脑子里迅速过了一遍时间线：
907年，后梁刚建立，朱温刚篡位，
唐朝正式灭亡，五代十国开始。

他喃喃自语："我他妈穿越到五代了。"

身后有人说："你说什么？"

李归舟回头，看到一个透明的人站在他身后，
穿着武将的衣服，胸口有一个血洞，
表情介于愤怒和困惑之间。

"你能看到我？"那人问。

林觉看了看手里的《酉阳杂俎》，
书页自动翻到了某一页，上面写着：
"执念深者，死而不散，留于阳间……"

李归舟叹了口气："好，我能看到你。
你叫什么名字，怎么死的，想要什么？"

那人愣了一下，然后说：
"郭长风。被人害死的。
 我想……我想让这个天下太平。"

李归舟沉默了一会儿。

郭长风看了他一眼："你姓什么？"
"李。"
郭长风沉默了片刻，眼神复杂：
"李……好，姓李的，先说说那个锁魂钉的事。"

（李归舟内心：他为什么这个表情。
 哦。唐朝国姓。我忘了。）
```

**第一卷主线**：
```
① 李归舟在汴州站稳脚跟
   - 用历史知识预判了一次小规模的兵变，
     救了一个商人，换来了落脚之地
   - 开始学习古代生存技能（骑马、用剑的基础）
   - 与郭长风磨合：两人经常吵架，
     但在关键时刻互相救了对方
   - 身份：自报"从关中逃来的流民"
     乱世中，来历不明的人太多，没人细究

② 第一根锁魂钉
   - 位置：汴州城内，某个废弃的官署地下
   - 守护者：一只被锁魂钉的邪气养大的厉鬼
             （原本是官署里冤死的小吏）
   - 化解方式：李归舟找到小吏的冤情，
               帮助申冤，厉鬼化解，锁魂钉暴露
   - 拔钉代价：李归舟手掌受伤，留下疤痕

③ 第一卷的感召人物：崔晚烟
   - 白马驿之祸（905年）的幸存者，
     在汴州以抄书为生
   - 她代表"文化的传承"——乱世中，
     她在默默抄写那些可能失传的典籍
   - 她对李归舟说：
     "你知道吗，我抄的这些书，
      将来可能是这个时代留下的唯一痕迹。
      历史不是皇帝写的，是这些字写的。"
   - 李归舟（内心）：这话放在现代，
     能发一篇核心期刊。
   - 崔晚烟抬头看了他一眼，没有多问，
     继续低头抄书。

④ 第一卷结尾
  - 第一根锁魂钉拔除，郭长风的魂魄稍微自由了一些
   - 他能短暂地"显形"让普通人看到（但很耗力）
   - 李归舟发现：郭长风的执念不只是"申冤"，
     他真的在乎这个天下
   - 李归舟第一次认真问郭长风：
     "你觉得，什么样的天下才算太平？"
   - 郭长风想了很久，说：
     "百姓能种地，孩子能读书，
      打仗的人不用打了，
      死了的人能入土。
      就这些，不多。"
   - 李归舟没说话，但他记住了这句话。
```

### 📖 第二卷：五钉之路（912—922年）

**主题**：走过这片土地，才知道它值得被守护

**五根锁魂钉的位置与对应故事**：

```
【第一钉】汴州（第一卷已完成）

【第二钉】洛阳（912—914年）
  位置：洛阳城外，白马寺附近的乱葬岗
  守护者：战场饿鬼群（梁晋战争的死亡士兵）
  故事核心：
    - 李归舟进入鬼域，发现饿鬼们不是在"作恶"，
      而是在"等"——等有人来收尸，等有人来祭祀
    - 李归舟用现代人的方式处理：
      找到附近村民，组织了一次集体祭祀
    - 饿鬼们得到祭祀，执念减弱，锁魂钉暴露
  感召人物：一个后梁的普通士兵（活人）
    - 他在战场上见过太多死，已经麻木了
    - 李归舟帮助他找到了战死兄弟的魂魄，
      让他们说了最后一句话
    - 士兵哭了，说："我以为他们恨我，
      因为我活下来了。"
    - 兄弟的魂魄说："我们不恨你，
      我们只是不想让你也死。"
  郭长风的反应：
    - 沉默很久，然后说：
      "我打了十年仗，从来没想过那些死在我身边的人
       在想什么。"

【第三钉】太原（915—917年）
  位置：太原城内，李克用的旧王府
  守护者：李克用的执念残影（非鬼魂，是执念的具象化）
  故事核心：
    - 李克用死后（908年），他的执念（三支箭的遗命）
      在旧王府中形成了一个"执念结界"
    - 锁魂钉被困在结界中心
    - 林觉需要进入结界，理解李克用的执念，
      才能解开结界
    - 林觉进入后，看到了李克用临死前的场景：
      他把三支箭交给李存勖，说的不是"复仇"，
      而是"我只是不想让我的人白死"
    - 林觉理解了：李克用的执念不是仇恨，
      是一个父亲对儿子的托付
  感召人物：年轻的李存勖（历史人物）
    - 李归舟在太原短暂见到李存勖
    - 李存勖问："你觉得我能灭梁吗？"
    - 李归舟（知道历史）："能。但灭梁之后呢？"
    - 李存勖愣了。
    - 李归舟（内心）：可惜你灭梁之后就废了，
      但这话我不能说。
  郭长风的反应：
    - 看到李克用的执念残影，郭长风久久不语
    - 最后说："他和我一样，
      都是死了还放不下。"
    - 李归舟："区别是，他的儿子能替他完成。
      你的呢？"
    - 郭长风沉默。

【第四钉】吴越（918—920年）
  位置：钱塘江边，一座古老的水神庙
  守护者：一条修炼了三百年的蛟，
          被锁魂钉的邪气污染，开始伤人
  故事核心：
    - 蛟本是钱塘江的守护者，性情温和
    - 被邪气污染后开始攻击渔船
    - 林觉查明真相，不是"斩蛟"，
      而是找到锁魂钉，拔除邪气，蛟恢复正常
    - 蛟为了感谢，帮助林觉渡过了一段危险的水路
  感召人物：吴越王钱镠（历史人物，此时已年老）
    - 李归舟见到了晚年的钱镠
    - 钱镠说了那句著名的话：
      "陌上花开，可缓缓归矣。"
    - 李归舟问："您不想统一天下吗？"
    - 钱镠："我只想让我治下的百姓，
      能慢慢走回家，不用跑。
      跑，是因为后面有兵。
      慢慢走，是因为家还在。"
    - 李归舟（内心）：这是我在这个时代，
      听到的最好的治国理念。
  郭长风的反应：
    - 郭长风第一次说：
      "我以前觉得，不统一天下就是失败。
       但我现在觉得，
       让一方百姓过上好日子，
       也算是赢了。"

【第五钉】燕云边境（921—922年）
  位置：燕云十六州边境，一座被废弃的烽火台
  守护者：最强的守护者——一个"活着的人"
          政敌的后人，继承了祖先的邪术，
          专门守护最后一根锁魂钉
  故事核心：
    - 这是最难的一关，因为对手是人，不是妖
    - 林觉不能用残卷对付人
    - 需要用智谋和郭长风的配合来解决
    - 最终，林觉揭露了政敌当年陷害郭长风的真相，
      让守护者的后人知道了祖先的罪行
    - 守护者的后人选择了放手：
      "我不能替祖先赎罪，但我可以不继续他的错。"
  感召人物：燕云边境的汉人老者
    - 老者说：
      "我这辈子，先是唐朝人，
       后是后梁人，后是后唐人，
       现在是契丹人。
       我只想知道，我到底是哪里的人？"
    - 李归舟想了很久，说：
      "您是这片土地上的人。
       不管它叫什么，这片土地是您的。"
    - 老者哭了。
    - 郭长风（在李归舟耳边）：
      "这句话，比我说过的所有豪言壮语都好。"
```

### 📖 第三卷：归处（922—927年）

**主题**：送你去你该去的地方，我不知道那是哪里，但你知道

**主线剧情**：
```
【五钉全部拔除（922年）】
  - 郭长风的魂魄完全自由，可以投胎了
  - 但问题出现了：
    投胎需要找到"合适的人家"
    不是随便一户都行——魂魄与人家之间有"缘分"
    郭长风感知到了那个地方，但说不清楚在哪里
    只知道：在北方，在一个小城，有一户普通人家

【寻找投胎之所（922—926年）】
  - 林觉带着郭长风的魂魄，在北方寻找那户人家
  - 这段旅程是全书最安静的部分：
    没有大战，没有妖怪，只有两个人走路
  - 路上的对话是全书最重要的部分：

  郭长风："你穿越过来，后悔吗？"
  李归舟："后悔什么？"
  郭长风："后悔遇到我。"
  李归舟："……你这个问题问得很蠢。"
  郭长风："怎么蠢了？"
  李归舟："因为如果没遇到你，
         我在这个时代就是一个孤魂野鬼。
         是你让我知道，这个时代值得被在乎。"
  郭长风沉默了很久。
  "那我也不后悔，"他说，
  "遇到你这个姓李的怪人。"

  ——

  郭长风："你说，我投胎之后，
           还会记得这些吗？"
  李归舟："不会。投胎之后，前世的记忆会消失。"
  郭长风："那我那些抱负呢？"
  李归舟："抱负不是记忆，是性格。
         你的性格会带过去的。"
  郭长风："你怎么知道？"
  李归舟："……我学历史的，我知道。"
  （李归舟内心：因为我知道你投胎之后是谁。）

  ——

  郭长风沉默了很久，然后说：
  "归舟。"
  李归舟："嗯？"
  郭长风："你这个名字，
           是要归到哪里去？"
  李归舟想了想，没有回答。
  郭长风也没有再问。
  两个人就这么走着。

【找到那户人家（926年末）】
  - 在河北涿州（一说洛阳），林觉找到了那户人家
  - 普通的农户，男主人是个小武官，
    妻子正怀着身孕
  - 郭长风的魂魄在那户人家门口停下来，
    说："就是这里。"
  - 林觉问："你确定？"
  - 郭长风："确定。我说不清楚为什么，
    但就是这里。就像……回家一样。"

【送别（927年初）】
  - 郭长风准备投胎
  - 临别前，他对林觉说：
    "你帮了我这么多，我没什么能给你的。
     只有一句话：
     你在这个时代，不是过客。
     你见过的那些人，你帮过的那些魂，
     都是真实的。
     你也是真实的。"
  - 林觉："……去吧。"
  - 郭长风的魂魄消散，化为一道光，
    飘入那户人家。

【揭晓（927年，赵匡胤出生）】
  - 李归舟站在那户人家门外，听到里面传来婴儿的哭声
  - 邻居来道喜，问："生了？男孩女孩？"
  - 里面的人说："男孩！"
  - 邻居："起名了吗？"
  - 里面的人说："他父亲说，
    就叫……匡胤。赵匡胤。"

  李归舟愣在原地。

  他想起郭长风说的那句话：
  "百姓能种地，孩子能读书，
   打仗的人不用打了，死了的人能入土。
   就这些，不多。"

  他想起钱镠说的：
  "让百姓慢慢走回家，不用跑。"

  他想起那个燕云老者的问题：
  "我到底是哪里的人？"

  他想起柴荣还没出生，
  想起赵匡胤还要再长三十三年才能黄袍加身，
  想起这个天下还要再乱很久……

  但他突然不那么焦虑了。

  因为他知道，
  那个抱负，
  已经种下去了。

  他低头看了看自己的手。
  手背上，是当年拔第一根锁魂钉留下的疤。

  他没有想太多。
  只是站在那里，听着婴儿的哭声，
  觉得这个世界还挺好的。

【结局】
  - 李归舟没有回到现代（或者说，他不知道怎么回去）
  - 他选择留在这个时代
  - 最后一幕：他坐在汴河边，
    手里还是那本《酉阳杂俎》
  - 书页翻到最后一页，上面只有一句话：
    "物老则群精依之，因人而见形。"
  - 李归舟笑了笑，把书合上。
  - 全书终。
```

## 九、文风示例

```
【李归舟的内心独白风格】
  "我他妈穿越到五代了。"
  "好，冷静，我是历史系的，我能处理这个。"
  "……我处理不了这个。"
  "而且我还姓李。在这个时代姓李。太好了。"

  ——轻快、自嘲，但不廉价

【郭长风的说话风格】
  "你这个姓李的现代人，说话怎么这么奇怪。"
  "什么叫'没信号'？"
  "你说的那个'赵匡胤'是谁？"
  "你姓李，但你不像那些姓李的。"

  ——直率、有点莽，偶尔一句话戳心

【两人的对话节奏】
  郭长风："你怕死吗？"
  李归舟："怕。"
  郭长风："那你还来？"
  李归舟："怕死和不去是两回事。"
  郭长风沉默了一会儿。
  "你这个姓李的，"他说，"有点意思。"

【历史知识的运用方式】
  不是"我知道历史所以我无敌"，
  而是"我知道大概会发生什么，
       但细节我不知道，
       而且历史是会被改变的"。
  ——保持紧张感，不让金手指变成作弊器
```

## 十一、妖魔鬼怪分类与处理方式

> 主角不是"降妖机器"，他是一个有判断力的现代人。
> 处理方式不固定——杀、封印、超度，要看妖魔鬼怪的性质。

### 🔴 恶——直接斩杀 / 封印
> **来源**：乱世中最坏的人死后，恶念不散，化为害人的妖魔

| 类型 | 来源 | 处理方式 | 收服后获得的能力 |
|------|------|----------|----------------|
| **煞气将**（战场屠将的执念） | 屠城将领死后，杀戮之气凝聚 | 斩杀——恶念无法化解，只能斩断 | 煞气感知（能感知周围的杀意） |
| **蛊主**（贪官的贪念） | 贪官死后，贪念化为蛊虫群 | 封印——蛊虫散不尽，只能封住 | 识蛊（能看穿被蛊虫控制的人） |
| **血尸**（暴君的怒念） | 暴君死后，怒气化为血尸 | 斩杀——纯粹的破坏力，无法沟通 | 怒气御（短暂借用极强的爆发力） |
| **锁魂者**（政敌的邪术） | 活人施邪术，死后术法仍在运作 | 封印术法根源，斩断邪术链条 | 解锁（能破解任何封印类邪术） |

### 🟡 中性——能化解就化解，不能就封印
> **来源**：普通人死后执念未散，或修炼中的精怪被乱世波及

| 类型 | 来源 | 处理方式 | 收服后获得的能力 |
|------|------|----------|----------------|
| **战场饿鬼** | 战死士兵，无人祭祀 | 超度——组织祭祀，让他们得到安息 | 鬼语（能与鬼魂短暂交流） |
| **冤鬼** | 冤死小吏，执念是申冤 | 化解——帮助申冤，执念自然消散 | 感知执念（能看到执念根源） |
| **钱塘蛟** | 修炼三百年的蛟，被邪气污染 | 拔除污染源，蛟自然恢复 | 水感（能感知水中异动） |
| **古树精** | 千年古木，被战火波及 | 安抚——找到伤它的根源，帮它愈合 | 生机（能感知生命力） |

### 🟢 善——不处理，甚至可以结盟
> **来源**：修炼有成的精怪，或守护一方的神灵，本身无害

| 类型 | 来源 | 处理方式 | 与主角的关系 |
|------|------|----------|------------|
| **水神**（河伯） | 古老水神，守护水域 | 不处理，礼敬即可 | 帮助主角渡过危险水路 |
| **山神**（燕云） | 燕云山脉古神 | 不处理，借道即可 | 提供庇护，指引方向 |
| **九尾天狐** | 修炼千年，已近神灵 | 不处理，平等相待 | 提供情报，预警危险 |

```
核心逻辑：
  主角不是"妖怪都该死"，
  也不是"妖怪都可怜"，
  而是：具体问题，具体分析。

  妖魔鬼怪是乱世最坏的人的映射——
  但不是所有妖魔都坏，
  就像不是所有人都坏一样。

  每收服一位妖魔鬼怪，解锁一个能力，
  从低级到高级，循序渐进。
```

## 十二、思想线设定——现代人的价值观如何影响那个时代

> 李归舟最大的"金手指"不是残卷，而是思想。
> 他带着教员等伟人的思想穿越回去，
> 随着自己的成长，言行一致，逐渐影响那个时代的人。

### 核心逻辑

```
李归舟不是穿越者中最能打的，
不是最聪明的，
但他有一样东西，那个时代没有人有：

他知道"人民"是什么。

不是"百姓"——百姓是被统治的对象。
不是"黎民"——黎民是需要被怜悯的群体。
而是"人民"——是历史真正的创造者。

这个认知，在五代乱世，
比任何武功都要危险，
也比任何武功都要有力量。
```

### 思想成长三阶段

**第一卷（907—912年）：说出来，但没人懂**

```
李归舟刚穿越，思想是现成的，但表达方式是错的——
他用现代语言说现代道理，没人听得懂。

场景示例：
  李归舟看到流民被驱赶，脱口而出：
  "这不对，人民才是历史的创造者——"

  郭长风：你在说什么？
  李归舟：……我是说，这些人不该被这样对待。
  郭长风：这有什么好说的，乱世就是这样。
  李归舟：乱世不是天灾，是人造的。
  郭长风沉默了一下：你这话，要是让朱温听到，
           你今天就死在这里。
  李归舟：……我知道。但这是真的。

特点：思想是对的，但李归舟还不会"说"——
他的话太超前，别人听不懂，或者听懂了也不敢信。
```

**第二卷（912—922年）：用行动说，开始有人跟**

```
李归舟不再直接讲道理，而是用行动示范。
他开始学会用那个时代的语言，说那个时代的人能听懂的道理。

不说"人民是历史的创造者"，
而说"这片土地是你们的，不是皇帝的"。

不说"阶级压迫"，
而说"为什么打仗的人最穷，
      种地的人最饿，
      读书的人最少？"

不说"群众路线"，
而说"你要治好这个地方，
      先去问问住在这里的人，他们要什么。"

特点：思想开始落地，开始有人被影响——
不是被"教育"，而是被"触动"。
```

**第三卷（922—927年）：言行一致，思想已经种下**

```
李归舟自己都没意识到，他说过的话、做过的事，
已经在那些人心里生了根。

场景示例：
  第三卷中，李归舟再次路过洛阳，
  遇到了当年那个士兵——他已经退伍，
  在村里开了一个小学堂，教孩子读书。

  李归舟：你怎么想到教书的？
  士兵：你当年说的，"死了的人能入土，
        孩子能读书"——我记住了。
        我不会打仗了，但我会教书。

  李归舟愣了很久。
  他想起那句话是郭长风说的，他只是转述了一遍。
  但它已经在这里生根了。

特点：言行一致本身，在那个时代就是一种革命。
```

### 思想来源与对应场景

| 思想来源 | 核心观点 | 对应场景 | 影响的人物 |
|---------|---------|---------|----------|
| **教员·群众路线** | "从群众中来，到群众中去" | 帮助吴越渔民，不替他们解决问题，而是帮他们自己解决 | 钱镠（"你这个方法，比我的幕僚聪明"） |
| **教员·人民战争** | "战争的伟力之最深厚的根源，存在于民众之中" | 燕云边境，告诉守将：你守不住这里，因为这里的人不知道自己在守什么 | 燕云老者、守将 |
| **教员·矛盾论** | "具体问题具体分析" | 处理妖魔鬼怪时，不用固定方式——杀、封印、超度，看情况 | 郭长风（"你怎么每次都不按套路来？"） |
| **教员·实践论** | "实践是检验真理的唯一标准" | 不相信"天命"，不相信"皇权神授"，只看这个人做了什么 | 李存勖（被触动，但没能坚持） |
| **鲁迅·立人思想** | "救救孩子" | 看到孩子在乱世中的处境，李归舟第一次真正愤怒 | 崔晚烟（"你是第一个为孩子愤怒的人"） |
| **范仲淹·先忧后乐** | "先天下之忧而忧" | 对郭长风说：你的抱负不是为了自己，这才是真的抱负 | 郭长风（深受触动） |

### 郭长风的思想成长弧线

```
初期：
  郭长风的抱负是"统一天下"——
  这是那个时代英雄的标准答案。

中期（被李归舟影响后）：
  郭长风开始问：统一之后呢？
  统一是手段，不是目的。目的是什么？

后期（投胎前）：
  郭长风说：
  "我以前想的是'我要统一天下'，
   现在我想的是'天下的人要过好日子'。
   这两句话，差了一个字，差了一个世界。"

——这就是李归舟的思想，种在了郭长风身上，
   然后随着郭长风，投胎进了那户姓赵的人家。
```
