# 主线暗线故事设定

> **状态**：三版（唐末乱世重构）  
> **核心道具**：造化丹  
> **时代背景**：唐末乱世——大唐已亡，天下大乱，人相食  
> **贯穿主题**：人心比妖更可怕；乱世中的侠义；废墟中的希望与绝望  
> **篇章结构**：序章 + 第一章 + 第二章 + 第三章（太平篇/终章）

---

## 一、世界核心设定

### 1.1 妖魔的起源（世界观根基）

> **核心公式：怨气/杀业 → 灵气污染 → 妖魔滋生**

- 盛唐时期：天地灵气清净，仙家维护秩序，妖魔极少
- 唐末乱世：人吃人、战乱、屠杀 → 大量怨气 → 灵气被污染 → 妖魔爆发
- 恶性循环：妖魔越多→杀戮越多→怨气越重→妖魔更多
- **造化丹是这个循环的"加速器"**——炼制过程产生极浓烈的怨气，大规模催生妖魔

### 1.2 仙家衰弱的原因

| 原因 | 说明 |
|------|------|
| **灵气被污染** | 仙家力量来源于清净灵气，污染后仙家衰弱 |
| **香火断绝** | 土地公/山神的力量来自百姓供奉，百姓死光/逃光，仙家消散 |
| **数量碾压** | 乱世制造的妖魔数量远超仙家能处理的上限 |
| **造化丹加速** | 有人在刻意加速灵气污染（暗线） |

### 1.3 造化丹的真相

| 层次 | 谁知道 | 知道什么 |
|------|--------|---------|
| **表层（民间）** | 百姓/江湖 | "造化丹能让人脱胎换骨" |
| **中层（修行者）** | 道士/僧人 | "造化丹是上古至宝，能大幅提升灵气感知" |
| **深层** | 千佛寺/秦宗权的宗教势力 | "需要活人炼制" |
| **真相** | 无人完全知晓 | **只有有灵根的人才能炼出来**（大量无灵根者被白白牺牲） |

> **关键暗线**：造化丹是这个世界怪物起源的线索——人吃人、世间大乱才造就了怪物，而造化丹的炼制在加速这个过程。

### 1.4 灵力的真相

- 这个世间的灵力**不足以飞升**
- 所谓"飞升成仙"从未有人做到过
- 但这个真相几乎无人知晓——大多数修行者还在追求飞升
- 国师可能隐约感知到了这一点（活了近二百岁仍未飞升），但他选择用另一种方式面对

## 二、核心人物设定

### 2.1 国师（终章核心人物）

| 项目 | 内容 |
|------|------|
| **身份** | 曾经的朝廷国师，正统道教传人 |
| **性别** | **女性** |
| **年龄** | 近二百岁——经历了盛唐到唐末的全过程 |
| **核心** | 她记得盛唐的美好，无法接受眼前的乱世废墟；也无法忘记师弟燕行烈 |
| **修为** | 极高——可能是当世最强的修行者之一 |
| **炼丹造诣** | 极高，曾管理朝廷所有丹药出品（朝廷还在时） |
| **信仰** | 正统道教，不杀生戒律——她不碰造化丹这种邪物 |
| **当前状态** | **已去世**——在故乡旧址用毕生法力造就"盛唐幻境"后耗尽法力去世，幻境仍在自动运转 |
| **悲剧** | 幻境是她死后留下的执念——一个已死之人的最后愿望在不断吞噬活人 |

**国师的故事线**：
```
盛唐时期：年少时随师傅来过京城，见过盛唐之景
  ↓
刻苦修行：与师弟燕行烈一同修行，斩妖除魔，炼丹证道，志同道合
  ↓
入朝为官：成为国师，管理丹药，辅佐皇帝
  ↓
师弟离世：燕行烈在一次斩妖中殉道（后被地府录用为鬼差）
  ↓
安史之乱：天下开始崩坏，她亲眼看着盛世瓦解
  ↓
唐末乱世：朝廷名存实亡，皇帝逃命，她是最后的支柱
  ↓
无力回天：无法停止节度使之间的战争杀戮，修为再高也救不了天下
  ↓
一生终点：斩妖除魔一生的终点时，想起儿时的美好回忆，想起师弟
  ↓
造就幻境：回到故乡旧址，用毕生法力重现记忆中的盛唐
  ↓
耗尽法力：造就幻境后，她耗尽毕生法力，去世
  ↓
幻境自运转：幻境在她死后依然自动运转——靠从外界汲取生命力维持
  ↓
制造尸潮：幻境不断从外界汲取生命力——制造了尸潮
  ↓
残留意志：幻境中有她年轻时的"影子"（她记忆中年轻的自己）
            幻境中的"妖鬼"扮演侠义除魔，清理尸潮
            一切都在自动运转——没有人在控制，只有执念在驱动
```

**国师的悲剧性**：
> 她不是恶人。她是一个**活得太久、看得太多、最终崩溃的老人**。
> 她用幻境逃避现实——就像一个老人反复翻看年轻时的照片，拒绝承认一切已经结束。
> 幻境中是她年轻时的影子——那是她记忆中最美好的时光。
> **但她已经死了。**幻境是她死后留下的执念——一个不愿停下的梦。
> 没有人在控制这个幻境——它只是在自动运转，不断吞噬路过的活人。
> 这比"一个活着的恶人"更让人悲哀——因为**连恨都找不到对象**。
> 她是这个故事里最悲哀的人。

### 2.2 "黄角式人物"（白莲教雏形的创始者）

| 项目 | 内容 |
|------|------|
| **身份** | 民间术士/散修，出身穷苦 |
| **核心** | 用符水赈灾、收容流民——在乱世中救人 |
| **手段** | 粟米放符水中分发给饥民（私自赈灾，违法） |
| **信众** | 穷苦百姓——他们加入不是因为信仰，而是因为饿 |
| **定位** | **白莲教的雏形**——一开始是善的，是穷苦百姓的一份救赎 |
| **与玩家** | 玩家在野外能遇到他的信众，可以帮助/交流 |

**设计意图**：
- 历史上白莲教的起源就是底层百姓的互助组织
- 这个角色代表"乱世中最朴素的善"——没有大道理，就是"人饿了要吃饭"
- 他的组织现在是善的，但暗示了未来可能被野心家利用变质
- 玩家可以选择帮助他们——这是一条温暖的支线
- **他不是反派，他是希望**

### 2.3 秦宗权（第二章核心反派/历史原型）

| 项目 | 内容 |
|------|------|
| **身份** | 军阀，控制中原南部 |
| **核心** | 四处吃人——军队以人肉为粮 |
| **与宗教的勾结** | 与某宗教势力合作，一边杀人吃人，一边炼化人尸（造化丹） |
| **定位** | 第二章的核心敌人——"人间地狱"的制造者 |
| **危险性** | 他不只是军事威胁，他的杀戮在大规模制造妖魔（灵气污染） |

**秦宗权 + 宗教勾结的逻辑**：
```
秦宗权的军队四处杀人吃人
  → 产生大量尸体和怨气
  → 与之勾结的宗教势力利用尸体/活人炼制造化丹
  → 炼制过程产生更多怨气 → 催生更多妖魔
  → 秦宗权得到"法术支持"（宗教势力提供的邪术增强军队）
  → 宗教势力得到"材料"和保护
  → 互利共生的邪恶循环
```

### 2.6 李克用 / 朱温（乱世枭雄/历史原型）

| 人物 | 势力 | 特征 | 与玩家 |
|------|------|------|--------|
| **李克用** | 沙陀军/河东 | 骁勇善战，亦正亦邪，与朱温死敌 | 中立/可合作 |
| **朱温** | 宣武军 | 野心勃勃，最终篡唐的枭雄 | 中立/敌对 |

**设计意图**：
- 他们是"世界背景"级别的存在——玩家能感受到他们的影响，但不一定直接对抗
- 他们代表"乱世的本质"——不是妖魔在毁灭世界，是人在毁灭世界
- 北境的高难度来自他们的战争——战场亡魂是最强的怨灵

### 2.7 燕行烈（国师师弟 / 鬼差 / 燕大侠）

| 项目 | 内容 |
|------|------|
| **身份** | 国师的师弟，曾经的讨魔校尉，现为地府鬼差 |
| **性别** | **男性** |
| **核心** | 生前与师姐（国师）志同道合，斩妖除魔；死后被地府录用，继续维持凡间秩序 |
| **生前** | 与师姐一同修行，侠肝义胆，在一次斩妖中殉道 |
| **死后** | 被地府录用为鬼差，在凡间维持阴阳秩序（与原著接轨） |
| **与国师** | 年轻时志同道合的师弟——国师造幻境的动机之一就是想回到和师弟在一起的时光 |
| **与玩家** | 战友关系——从冲突到并肩作战到信任 |
| **定位** | 类似原著讨魔校尉燕的角色——在夜间活动，斩杀失控妖鬼，身份神秘 |
| **关键信息** | **燕大侠不知道师姐已死**——他身为鬼差不了解人间近况 |

**燕行烈的出场节奏**：

| 篇章 | 出场方式 | 与主角关系 |
|------|---------|------------|
| **第一章** | 露脸，与主角产生冲突（身份不明，行为可疑） | 对立/误解 |
| **第二章** | 与主角一起战斗，逐渐建立信任；末尾建议主角去朝廷京城找师姐 | 战友/并肩 |
| **第三章** | 待定 | 待定 |

**第一章的冲突**：
- 燕行烈以鬼差身份在夜间活动，斩杀即将失控的妖鬼
- 主角不知道她的身份，只看到一个神秘女子在"杀人"（实际是杀即将变妖的人）
- 两人产生冲突——主角认为她在滥杀无辜
- 后来发现真相：她杀的"从来不是人"——是即将变成妖怪的人
- 冲突化解，但她不愿多说身份，消失

**第二章的并肩**：
- 在追查造化丹的过程中再次相遇
- 她也在追查——因为造化丹在大规模制造妖鬼，增加了她的"工作量"
- 两人合作战斗，逐渐建立信任
- 主角开始了解她的过去——她提到"师兄"（国师），但不说太多
- 感情线萌芽

**第三章的幻影**：
- 国师的幻境中有燕行烈的"影子"——那是国师记忆中年轻时的她
- 幻影中的燕行烈在帮主角斩妖除魔（清理尸潮）
- 但这不是真正的燕行烈——是国师的记忆投影
- 真正的燕行烈（鬼差）可能也会出现——面对师兄的幻境，她会怎么选择？
- 情感高潮：主角、真实的燕行烈、国师——三人之间的纠葛

**燕行烈的悲剧性**：
> 他死后依然在做生前的事——斩妖除魔，维持秩序。
> 他的师姐（国师）为了"回到和他在一起的时光"造就了幻境——然后死了。
> 但他不知道师姐已死——身为鬼差，他不了解人间近况。
> 他还以为师姐还在朝廷京城——所以建议主角去找她。
> 当他最终得知真相时……
> 他是这个故事里最坚韧的人。

## 三、篇章结构

### 3.1 总览

| 篇章 | 名称 | 区域 | 核心内容 | 黄皮书目标 |
|------|------|------|---------|-----------|
| **序章** | （待定） | 江南·开局村落 | 入门、学习基础、遭遇画皮鬼（必败） | 目标1：画皮鬼 |
| **第一章** | （待定） | 江南 | 成长→击败画皮鬼→线索指向千佛寺 | 目标1续：画皮鬼（二战，吃了造化丸强化） |
| **第二章** | （待定） | 江南·千佛寺 | 追查造化丸→揭露千佛寺→发现与秦宗权勾结 | 目标2：千佛寺Boss |
| **第三章** | 太平篇 | 国师故乡 | 进入盛唐幻境、揭露国师真相、最终抉择 | 最终目标：幻境/国师残留意志 |

### 3.2 序章——初入乱世

**地点**：江南·偏远村落

**故事**：
- 主角凭空出现（黄皮书选中），身处一个偏远村落
- 村民告知基本世界常识："天下大乱""朝廷没了""外面在打仗"
- 导师NPC（类似刘老道）出现，教授基础
- 村落中出现画皮鬼——从农家吃新郎开始作祟
- 黄皮书上显示目标：画皮鬼
- **第一战：必败**——画皮鬼太强，主角实力不足，被击败/逃脱
- 画皮鬼逃离村落，消失在远方
- 导师指引方向：去越州城，那里有钱镠的军队维持秩序，先成长再追踪画皮鬼

**序章结束时玩家知道**：
- 这是一个乱世
- 有妖魔鬼怪（画皮鬼是黄皮书的目标）
- 自己现在实力不足，需要成长
- 画皮鬼逃走了——它还会再出现
- 有人在维持秩序（钱镠），有人在作恶

---

### 3.3 第一章——画皮鬼

**地点**：江南（越州城、泉州、玉衡山、荒山野外）

**黄皮书目标**：画皮鬼（主线妖物之一）

**主线推进**：
```
越州城落脚 → 接委托除妖赚钱、学艺成长、慢慢提升实力
  ↓
【画皮鬼线】
  → 画皮鬼从农村逃到城镇，伪装成权贵（安全地获取猎物）
  → 城镇中出现权贵行为异常的传闻（失踪案、诡异宴会）
  → 主角调查，识破画皮鬼的伪装
  → 第二战：画皮鬼比序章时更强——它吃了千佛寺的造化丸
  → 击败画皮鬼（第二战胜利）
  → 关键线索：画皮鬼身上/巢穴中发现造化丸残留
  → 线索指向造化丸的来源——千佛寺地界
  → 第一章结束，黄皮书翻页，显示下一个目标
```

**支线内容**：
- 玉衡山·山宴（妖界社交）
- 蛇头山·山贼巢穴（战斗副本）
- "黄角式"组织的支线（帮助穷苦百姓）
- 各种志怪委托（赚钱、提升实力）
- 食人者的陷阱（恐怖事件）
- **燕行烈露脸**——夜间遭遇神秘女子"杀人"，产生冲突（见2.7节）

**第一章结束时玩家知道**：
- 画皮鬼已被击败，它变强是因为吃了造化丸
- 造化丸来源指向千佛寺
- 有一个神秘女子（燕行烈）在夜间活动，身份不明
- 下一步：去千佛寺追查造化丸的来源

---

### 3.4 第二章——千佛寺

**地点**：江南·千佛寺及周边

**黄皮书目标**：千佛寺Boss（主线妖物之二）

**主线推进**：
```
追查造化丸来源，前往千佛寺地界
  → 从委托/NPC中了解到千佛寺的异常
  → 调查千佛寺（圆通提供线索）
  → 发现千佛寺在做"肉身佛"——活人炼制
  → 深入千佛寺，发现与军阀（秦宗权残部）的勾结
  → 千佛寺 + 秦宗权残部 = 造化丹的炼制工坊
  → 一边杀人吃人，一边炼化人尸
  → 击败千佛寺Boss
  → 获得关键线索：
    - 造化丹需要活人炼制，只有有灵根者才能成功
    - 造化丹的炼制在大规模污染灵气、催生妖魔
  → 燕大侠面对大量造化丹没头绪
  → 燕大侠建议："师姐在世的话应该会很了解，去朝廷京城看看"
  → 燕大侠不知道师姐已死——他是鬼差，不了解人间近况
  → 玩家前往朝廷京城寻找"国师"（师姐）
  → 到了京城才得知国师已去世——"她最后回故乡去了""再也没回来"
  → 获得国师留下的信，线索指向她的故乡
```

**支线内容**：
- **燕行烈的并肩线**（再次相遇，合作战斗，感情线萌芽）
- 千佛寺周边的恐怖探索（肉身佛、僧鬼、怨煞）
- 流民救助（与"黄角式"组织的再次相遇）
- 正统佛教僧人的支线（对千佛寺的痛心）

**第二章结束时玩家知道**：
- 千佛寺在炼制造化丹——需要活人，与秦宗权残部勾结
- 造化丹是妖魔爆发的"加速器"——暗线揭开
- 千佛寺被击败（或重创）
- 面对大量造化丹，燕大侠也没头绪
- **燕大侠建议**："师姐在世的话应该会很了解，去朝廷京城看看"
- 燕大侠不知道师姐已死——他身为鬼差不了解人间近况
- 线索指向朝廷京城——去找"国师"（师姐）

### 3.5 第三章·太平篇——盛唐幻梦

**地点**：国师故乡旧址

**主线推进**：
```
根据燕大侠建议前往朝廷京城 → 得知国师已去世 → 获得国师留下的信 → 前往国师故乡
  → 进入后发现：这里是"盛唐"——繁华街道、安居百姓、歌舞升平
  → 一切都太美好了——与外面的乱世形成强烈对比
  → 逐渐发现不对劲：
    - "百姓"的行为有些僵硬/重复
    - 街道的边缘在模糊/消散
    - 有"妖鬼"在扮演侠义除魔——帮主角清理"入侵的尸潮"
    - **有一个年轻女子的幻影在帮主角斩妖除魔**——那是国师年轻时的样子
  → 深入探索，发现幻境的边界——边界之外是废墟和尸潮
  → 真相揭开：
    - 这一切都是国师的幻境——用毕生法力重现记忆中的盛唐
    - 幻境中的"年轻女子"是国师记忆中年轻时的自己——不是活人
    - 幻境中的"妖鬼"是国师召唤来维持秩序的
    - 尸潮是幻境造就的——维持幻境需要从外界汲取生命力
    - **国师已经死了**——她在造就幻境后耗尽法力去世
    - 幻境是她死后依然在自动运转的"遗产"——一个已死之人留下的执念
    - 幻境中的"国师"只是她的残留意志/记忆投影——不是活人
  → 面对国师的残留意志：
    - 它不是活着的国师——是一段执念
    - 它还在重复着国师生前的话："这是最后的盛唐……师弟还在这里……"
    - 但它已经没有意识了——只是一个不愿停下的幻影
  → 最终Boss战 / 最终抉择
```

**太平篇的核心体验**：

| 阶段 | 玩家感受 |
|------|---------|
| 刚进入幻境 | 震撼——"原来盛唐是这样的" |
| 发现异常 | 不安——"这里有什么不对" |
| 揭露真相 | 悲凉——"原来这一切都是假的" |
| 面对国师 | 复杂——"他不是恶人，但他在害人" |
| 最终抉择 | 沉重——"没有完美的答案" |

**最终抉择（待设计）**：
- 打破幻境？（摧毁国师最后的执念，幻境消散，尸潮停止）
- 让幻境继续？（不忍心打破，但活人会继续被吞噬）
- 找到某种方法让幻境"安息"？（不是打破，而是让它平静地消散——最难但最好的结局？）

## 四、暗线设计

### 4.1 造化丹暗线（贯穿全局）

```
序章：玩家遭遇画皮鬼（必败），不知道造化丹的存在
  ↓
第一章：画皮鬼逃到城镇变强了——它吃了造化丸
        击败画皮鬼后，发现造化丸残留，线索指向千佛寺
  ↓
第二章：前往千佛寺追查，发现千佛寺在炼制造化丹
        千佛寺与秦宗权残部勾结——一边杀人吃人，一边炼化人尸
        造化丹是妖魔爆发的"加速器"——人吃人造就怪物的暗线揭开
  ↓
第三章：国师的幻境也在消耗生命力——与造化丹的逻辑相似
         （不是造化丹，但本质相同：牺牲活人维持某种"美好"）
         但国师已死——这是一个无人控制的悲剧
```

### 4.2 妖魔起源暗线

```
序章：妖魔就是妖魔，打就完了
  ↓
第一章：开始注意到——千佛寺周边妖魔特别多
  ↓
第一章末：隐约感觉到杀戮与妖魔有关
  ↓
第二章：从正统教派了解到"灵气污染"的概念
  ↓
第二章中：在秦宗权控制区亲眼看到——杀戮最多的地方妖魔最多
  ↓
第二章末：完整理解——怨气/杀业污染灵气→催生妖魔→恶性循环
  ↓
第三章：国师的幻境是"最后的净土"——但维持它的代价是制造更多死亡
```

### 4.3 国师暗线

```
第一章：玩家可能听说过"国师"这个人（NPC提及——"曾经有个国师，活了快两百岁"）
        燕行烈露脸但身份不明——玩家不知道他与国师的关系
  ↓
第二章：千佛寺击败后，面对大量造化丹，燕大侠也没头绪
        燕大侠建议："师姐在世的话应该会很了解，去朝廷京城看看"
        燕大侠不知道师姐已死——他身为鬼差不了解人间近况
        玩家前往朝廷京城寻找"国师"
  ↓
第三章：到了京城才得知国师已去世——"她最后回故乡去了""再也没回来"
        追踪到国师故乡，进入幻境
        幻境中有国师年轻时的幻影——揭露一切
```

## 六、剧情推动机制

### 6.1 黄皮书驱动

| 机制 | 说明 |
|------|------|
| **目标显示** | 黄皮书上显示下一个要解决的目标（妖物/Boss） |
| **线索解锁** | 完成委托/任务会解锁通往目标的线索 |
| **篇章结束** | 每击败一个黄皮书目标 = 一个篇章结束 |
| **自由度** | 玩家不必直奔目标——可以自由探索、接委托、赚钱 |

### 6.2 任务/委托系统

| 类型 | 来源 | 功能 |
|------|------|------|
| **悬赏委托** | 城镇告示栏 | 赚钱、提升实力、了解世界 |
| **NPC委托** | 特定NPC | 推进支线、解锁线索 |
| **随机遭遇** | 路途中 | 丰富体验、获取材料 |
| **主线任务** | 黄皮书/关键NPC | 推进主线、解锁新区域 |

### 6.3 信息获取方式

玩家通过以下方式获取推进主线的信息：

| 方式 | 示例 |
|------|------|
| **NPC对话** | "听说千佛寺最近不太对劲……" |
| **告示栏** | 悬赏告示中暗含线索 |
| **委托完成后** | 委托人透露更多信息 |
| **探索发现** | 在野外/副本中发现文书/证据 |
| **妖界情报** | 山宴上妖怪们的闲聊 |
| **通幽感知** | 主角的特殊能力——感知残留记忆/怨气 |

## 七、关键NPC对话设计

### 7.1 国师（第三章·太平篇·幻境中的残留意志）

**进入幻境后，遇到"国师"的影子**：
> 老槐树下。一个白发苍苍的老妇人坐在石凳上。
> 
> 她看起来很安详。像是在等什么人。
> 
> "……你来了。"
> 
> 她给你倒了一杯茶。茶是温的。但你知道——这不是真的茶。
> 
> "你看，这棵树。我小时候就在这棵树下玩。那时候……街上有卖糖人的，有耍猴的，有唱戏的。"
> 
> "师弟也在。他总是笑。"
> 
> 她的眼神有些空洞。像是在看着很远的地方。
> 
> "……我是不是已经死了？"
> 
> 她似乎在问你。又似乎在问自己。
> 
> "没关系。只要这里还在……他就还在。"
> 
> 她重复着这句话。一遍又一遍。
> 
> 你知道——这不是活着的国师。这只是一段执念。一个不愿醒来的梦。

**国师留下的信（从朝廷京城NPC处获得，第三章初）**：
> 一封泛黄的信。字迹苍老但有力。
> 
> "吾一生斩妖除魔，证道炼丹，辅佐帝王。"
> "然天下已乱，生灵涂炭，吾无力回天。"
> "师弟已去多年。吾亦将去。"
> "唯有此地——吾之故乡，吾与师弟年少时的故乡。"
> "吾以毕生法力，重现旧日之景。"
> "若有人来此——请看一眼。看一眼盛唐是什么样子。"
> "然后……随你处置吧。"

### 7.3 "黄角式人物"的信众（江南·野外）

**遇到信众的营地**：
> 几十个衣衫褴褛的人围坐在一起，面前是一锅稀粥。
> 
> 一个中年男人在分粥。粥里有一股淡淡的符水味。
> 
> "道长？来喝碗粥吧。不收钱。"
> 
> "我们这些人……都是从北边逃过来的。家没了，地没了，什么都没了。"
> 
> "师父说，人活着就得互相帮衬。他教我们画符，符水放粥里，能驱邪避瘟。"
> 
> "官府说我们是'妖人聚众'，要抓我们。可我们就是想活着啊。"
> 
> "道长，你不会告发我们吧？"

### 7.4 燕行烈（第一章·冲突）

**第一次相遇（夜间，主角目睹他"杀人"）**：
> 月光下，一个黑衣男子提剑而立。他脚下躺着一个"人"——但那"人"的身体正在扭曲变形。
> 
> 你冲上去拦住他。
> 
> 他看了你一眼。眼神冰冷。
> 
> "让开。"
> 
> "你不知道你在做什么。"
> 
> 你不让。他叹了口气。
> 
> "你看清楚——那不是人。"
> 
> 地上的"人"已经完全变形了。四肢扭曲，面目全非，发出野兽般的嘶吼。
> 
> 他一剑斩下。干净利落。
> 
> "我杀的，从来不是人。"
> 
> 他转身离去。没有解释更多。
> 
> "别跟着我。下次再挡路——我不会手下留情。"

### 7.5 燕行烈（第二章·并肩）

**再次相遇**：
> 他看到你时，微微皱眉。
> 
> "又是你。"
> 
> "……你也在查造化丹的事？"
> 
> 他沉默了一会儿。
> 
> "那东西在把人变成妖。我的活儿越来越多了。"
> 
> "既然目标一样……走吧。两个人总比一个人快。"
> 
> 他走在前面，没有回头。但你注意到——他的步伐放慢了一点，等着你跟上。

**第二章末尾（千佛寺击败后，面对大量造化丹）**：
> 他看着千佛寺地下堆积如山的造化丹，眉头紧锁。
> 
> "这东西……我没见过。"
> 
> "但师姐在世的话，应该会很了解。她炼丹造诣极高，朝廷的丹药都是她管的。"
> 
> "去朝廷京城看看吧。师姐应该还在那里。"
> 
> 他的语气很平常。你能感觉到——他真的不知道师姐已经死了。
> 
> 他是鬼差，不了解人间近况。
