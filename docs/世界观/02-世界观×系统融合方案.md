# 世界观 × 系统融合方案

> **状态**：已确认方向，待逐系统落地  
> **原则**：不动现有系统的代码架构，只调整设计层面的内容定义  
> **依据**：世界观总纲 + 地煞古代线提炼

---

## 一、修炼系统改造

### 1.1 去掉境界，改为"灵气感知"

| 项目 | 改造前 | 改造后 |
|------|--------|--------|
| 核心属性 | 修仙境界（练气→筑基→金丹…） | **灵气感知**（连续数值，无明确分界线） |
| 属性名称 | 统一叫法 | **各教派叫法不同**（见下表） |
| 提升方式 | 突破境界 | 学习不同属性类功法来提升 |

**各势力对"灵气感知"的称呼**：

| 势力/教派 | 称呼 | 修行者 |
|-----------|------|--------|
| 道家 | 道行 | 道士、方士 |
| 佛家 | 佛力 / 功德 | 僧人、行脚僧 |
| 民间散修 | 法力 / 术法 | 术士、阴阳先生 |
| 朝廷 | 视来源而定 | 国师、钦天监 |

> 本质都是同一个属性"灵气感知"，只是表现层的名字不同。

### 1.2 术法获取方式

| 获取渠道 | 说明 | 对应系统 |
|----------|------|---------|
| **黄皮书（核心道具）** | 击败对应Boss后解锁关键术法 | 类似现有的"绝技"系统 |
| **拜师学习** | 找到NPC师父，花费时间/金钱学习 | NPC交互系统 |
| **花钱购买** | 民间技能可以从商人/术士处购买 | 物品/交易系统 |
| **实战领悟** | 特定条件下在战斗中领悟（低概率） | 战斗系统触发 |

### 1.3 灵气感知的提升

- 通过学习**不同属性类功法**来提升灵气感知总值
- 功法分类保留五行体系作为底层（金木水火土）
- 学习某一属性的功法 → 该属性的灵气感知提升 → 总灵气感知提升
- 灵气感知越高 → 能学习/使用的术法越强

## 二、炼丹系统 → 制符系统

### 2.1 核心置换

| 项目 | 改造前 | 改造后 |
|------|--------|--------|
| 系统名称 | 炼丹系统 | **制符系统** |
| 制作物 | 丹药 | **符箓**（一次性消耗品） |
| 配方 | 丹方 | **符式**（符箓的画法） |
| 制作过程 | 炼丹炉 + 药材 | **朱砂 + 黄纸 + 法力** |
| 制作技能 | 炼丹术 | **画符术**（熟练度影响成功率） |

### 2.2 获取渠道置换

| 物品 | 改造前获取方式 | 改造后获取方式 |
|------|---------------|---------------|
| **符箓** | 商店购买/任务奖励 | **玩家自己制作**（制符系统） |
| **丹药** | 玩家自己炼制 | **商店购买/任务奖励/交易获得** |

> 简单说：符箓和丹药的"自制"与"购买"渠道互换了。

### 2.3 制符系统设计要点

- **符式获取**：师承（NPC教）、古籍（探索获得）、领悟（特定条件）
- **制作材料**：朱砂、黄纸、特殊墨（不同符需要不同材料）
- **制作过程**：消耗法力 + 时间，有成功率（受画符术熟练度影响）
- **符箓用途**：战斗中触发连携、探索中感知妖鬼、任务中超度/封印

## 三、战斗系统调整

### 3.1 核心战斗循环（保持不变）

> 战斗的核心玩法保持原有设计：**符箓触发连携 → 释放绝技**

- 第一个绝技式：剑类技能（主角核心战斗手段）
- 符箓作为连携触发器（制符系统产出 → 战斗系统消耗）

### 3.2 敌人设计分级

| 敌人级别 | 技能设计 | 说明 |
|----------|---------|------|
| **小怪** | 技能可重复 | 同类型小怪共享技能池，节省设计量 |
| **大怪（同类型）** | 进化/升级技能 | 同类型的大怪，后出现的技能是前者的升级版 |
| **Boss（独特）** | 单独设计技能 | 每个Boss有独特的技能组，不与其他敌人共享 |

### 3.3 战后处置系统（新增）

战斗目标统一为**击败**，击败后进入**处置选择**：

| 处置方式 | 效果 | 适用对象 |
|----------|------|---------|
| **驱赶** | 妖/鬼离开此地，可能日后再遇 | 善恶不明的妖、低威胁鬼物 |
| **超度** | 鬼物转世，永久消除，获得功德 | 有执念的鬼（怨灵、游魂等） |
| **封印** | 妖/鬼被封印，可作为材料或道具 | 强大但不想杀的妖物 |
| **谈判** | 妖/鬼成为盟友或提供信息/报酬 | 有智慧的妖、善妖 |

> 不同处置方式影响：声望、后续剧情、物品获取、NPC关系

## 四、NPC交互系统调整

### 4.1 NPC类型（基于世界观）

| 类型 | 原型 | 功能 |
|------|------|------|
| **导师型** | 刘老道 | 教授术法、提供指引、前期保护 |
| **战友型** | 燕行烈 | 协同战斗、提供情报、互相帮助 |
| **线人型** | 圆通 | 提供情报、带路、有自己的小算盘 |
| **委托人** | 村民/富户 | 发布任务、提供报酬 |
| **商人型** | 各类商贩 | 买卖物品、丹药、材料 |
| **神秘型** | 葫芦书生/衡先生 | 关键信息、隐藏任务、世界观深层 |

### 4.2 "恩义"系统

- 救人、帮忙、超度 → 建立恩义关系
- 恩义影响：NPC是否愿意帮你、是否提供折扣、是否透露秘密
- 恩义可以被消耗（求人办事）
- 参考原文：老道救了李长安 → 李长安跟随学习 → 李长安反过来保护老道

## 五、门派声望 → 江湖名声

### 5.1 多维度名声

| 维度 | 提升方式 | 影响 |
|------|---------|------|
| **百姓名声** | 除妖、救人、不收穷人钱 | 百姓主动提供线索/帮助 |
| **官府名声** | 帮官府办事、接悬赏 | 获得官方支持/度牒/通行 |
| **教派名声** | 帮助道观/寺庙、展现修为 | 获得师承/功法/法器 |
| **江湖名声** | 行侠仗义、以弱胜强 | 江湖人士主动结交/挑战 |
| **妖界名声** | 放过善妖、参加山宴 | 妖怪不主动攻击/提供情报 |

### 5.2 名声的双面性

> 名声大了，有人来找你帮忙，也有人来找你麻烦。

- 百姓名声高 → 更多委托找上门
- 官府名声高 → 官府可能强制征召你
- 江湖名声高 → 有人来挑战/报仇（参考原文：蛇头山杀贼后被刺客追杀）
- 妖界名声高 → 某些妖王视你为威胁

## 七、剧情系统 → 志怪事件驱动

### 7.1 事件结构模板

每个"志怪事件"是一个完整的故事单元：

```
【事件名称】
├── 触发条件（到达某地/接受委托/随机遭遇）
├── 调查阶段（收集线索、询问NPC、使用冲龙玉追踪）
├── 真相揭露（发现妖/鬼的来历和动机）
├── 对决阶段（战斗）
├── 处置选择（驱赶/超度/封印/谈判）
└── 后果影响（名声变化、NPC关系、后续事件触发）
```

### 7.2 事件规模分级

| 规模 | 特征 | 时长 |
|------|------|------|
| **小事件** | 单个妖/鬼，一个地点 | 1-2场战斗 |
| **中事件** | 多个线索，跨地点 | 3-5场战斗 |
| **大事件** | 多方势力，区域级影响 | 主线章节 |

## 八、记忆系统 → "通幽"感知

### 8.1 通幽能力

- 主角天生能看到鬼、触碰鬼（黄皮书赋予）
- 到达特定地点时，可以感知残留的记忆/怨气
- 通过"通幽"了解事件真相、鬼的来历、隐藏线索
- 是推动剧情和调查的核心能力

### 8.2 通幽的游戏表现

| 场景 | 表现 |
|------|------|
| 到达有鬼的地点 | 画面/音效变化，提示玩家此处有异 |
| 与鬼对话 | 普通人看不到的NPC，只有主角能交互 |
| 感知过去 | 播放"记忆片段"——过去发生的事件 |
| 追踪线索 | 看到常人看不到的痕迹（血迹、怨气、脚印） |

## 附：关于"地图设计"的说明

> 你说"地图设计没看懂"，这里重新解释：

之前提到的"圈层式地理设计"不是说游戏地图要做成同心圆，而是说**故事发生的地点按规模递进**：

```
开局：偏远村落（小地图，几个NPC，一个志怪事件）
  ↓
中期：小城/州城（中等地图，多个NPC，多条线索交织）
  ↓
后期：大城/重镇（大地图，多方势力，复杂事件）
```

**具体到游戏里**：
- 不是开放世界，不需要一个巨大的无缝地图
- 而是**一个个独立的"场景/地点"**，通过剧情/任务驱动玩家前往下一个
- 每个地点有自己的NPC、事件、可探索区域
- 类似《活侠传》或《侠客风云传》的地点切换模式
