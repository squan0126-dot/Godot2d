# 世界观总纲

> **状态**：当前生效版本  
> **调性定位**：中国志怪 + 侠义叙事（轻玄幻、重人情）  
> **参考作品**：《地煞七十二变》《倩女幽魂》《聊斋志异》《搜神记》  
> **注意**：本文档替代旧版修仙世界观设定（01-genesis 等），旧文档暂存不删

## 一、基础设定

| 项目 | 内容 |
|------|------|
| 时代背景 | 架空古代中国（不锁定朝代，可参考各历史阶段人物事件原型） |
| 能力上限 | 筑基以下（会几招法术 = 了不起的人物） |
| 民间认知 | 没有"修仙体系"概念，会术法者在百姓眼中即为仙人/妖人/术士 |
| 组织形态 | 凡间组织（朝廷、教派道观寺庙、民间帮派等），非修仙宗门 |
| 故事氛围 | 降妖除魔、恩怨情仇、鬼怪传说、江湖义气 |

## 二、侠义内核

> **侠之大者，为国为民；洒脱肆意；为不公、为弱小，挺身而出。**

| 关键词 | 表现 |
|--------|------|
| 为国为民 | 主角的行动不是为了修炼变强，而是因为"该做的事" |
| 洒脱肆意 | 不受门规/宗门/师命束缚，我行我素但有底线 |
| 为不公挺身而出 | 看到恶事不能忍，哪怕对手比自己强也要上 |
| 为弱小挺身而出 | 保护普通人、小人物，不是"顺手"而是"本能" |

核心驱动力：**"我不是因为能赢才出手，而是因为不出手就不是我。"**

## 三、超自然规则

### 3.1 力量来源：灵气感知

天地间存在灵气，少数人能感知并修炼。各家传承不同，叫法不同，本质相同。

| 体系 | 称呼 | 修行者 |
|------|------|--------|
| 道家 | 道行 | 道士、方士 |
| 佛家 | 佛力 / 功德 | 僧人、行脚僧 |
| 民间散修 | 法力 / 术法 | 术士、阴阳先生 |
| 朝廷 | 视来源而定 | 国师、钦天监 |

### 3.2 非人存在

| 类型 | 来源 | 特征 |
|------|------|------|
| **鬼** | 怨气 / 执念 / 凡间事未了 | 依附某地某物，有未竟之事 |
| **妖** | 非人生物修炼成精 | 狐、蛇、树、石等，有善有恶 |
| **僵尸** | 尸体异变 | 凶煞之物 |
| **仙家** | 土地公、山神等 | 维护一方平安的低级神职 |

### 3.3 飞升成仙

- **作为悬念存在**
- 传说中有人飞升成仙，但无人证实
- 朝廷高层了解"灵力有限""可以飞升"等说法，但不一定是真的
- 这是世界观的天花板悬念，不做定论

## 四、权力结构

```
┌─────────────────────────────────────────────┐
│            天（飞升成仙？未知）               │
│         传说中存在，无人证实                   │
├─────────────────────────────────────────────┤
│          朝廷高层 / 国师                      │
│  了解部分"世间真相"（灵力有限、飞升传说等）    │
│  性质：谋权谋财，利用术法巩固地位              │
│  有人甚至利用妖怪进行杀戮修炼，视百姓为玩物    │
├─────────────────────────────────────────────┤
│          教派（道观 / 寺庙）                   │
│  性质：修行为本                               │
│  部分修行者出手降妖除魔，是民间主要正义力量     │
│  到了朝廷性质就变了——为谋权谋财               │
├─────────────────────────────────────────────┤
│          民间                                 │
│  不了解修仙体系，会术法 = 仙人                 │
│  偏远地区妖怪横行，百姓苦不堪言                │
│  土地公 / 山神维护一方，但能力有限              │
├─────────────────────────────────────────────┤
│          偏远村落 / 荒野                       │
│  妖怪横行，无人管，自生自灭                    │
└─────────────────────────────────────────────┘
```

### 核心矛盾

- 有能力的人（朝廷/高位修士）→ 不管百姓死活，甚至加害
- 教派修行者 → 有心无力，或各有立场
- 百姓 → 苦于妖邪，无人庇护
- **主角的位置** → 正是在这个"没人管"的缝隙中，选择出手

## 五、信息迷雾（认知层次）

| 谁 | 知道什么 |
|----|---------|
| 百姓 | 只知道有鬼有妖，会法术的是仙人 |
| 教派修士 | 知道灵气修行体系，知道妖鬼本质 |
| 朝廷高层 | 了解"灵力有限""飞升传说"等更深的真相（但不一定是真的） |
| 主角（玩家） | 一开始什么都不知道，从村民口中逐步了解 |

> 玩家的认知和村民同步开始，随着冒险逐渐接触更深层的真相——这本身就是叙事钩子。

## 六、主角设定

| 项目 | 内容 |
|------|------|
| 名字 | 玩家自定义 |
| 身份 | 穿越者，凭空出现于异世界（无起源设定） |
| 世界认知 | 外乡人（世界里的人只当他是外乡人） |
| 初始能力 | 剑术（本质是"剑修"入门，但主角不知道） |
| 成长方向 | 从纯剑技逐步接触灵气感知，习得术法（但不自知体系） |

## 七、开局设计

| 项目 | 内容 |
|------|------|
| 落点 | 偏远村落 |
| 初始事件 | 村落有志怪作祟（具体事件待设计） |
| 世界观引入 | 通过村民口述了解基本常识（有妖有鬼、有道士除妖、远处有朝廷等） |
| 主角表现 | 以剑术解决志怪，展现"侠"的本能——看到不平事，出手 |
| 叙事目的 | 让玩家以"一无所知的外乡人"视角自然融入世界 |

---

## 八、设计原则（红线）

1. **不做完整修仙境界体系** —— 筑基以下即为天花板
2. **不做修仙宗门** —— 所有组织都是凡间形态
3. **不做开放世界刷刷刷** —— 事件驱动，非成长驱动
4. **法术是手段不是目的** —— 出手的理由永远是"人"
5. **世界观天花板留悬念** —— 飞升成仙不做定论
6. **主角的剑术暗含剑修本质** —— 但主角不知道，作为隐藏线索
