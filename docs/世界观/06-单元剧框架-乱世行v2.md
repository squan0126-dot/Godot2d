# 《乱世行》单元剧框架 v2

> **类型**：古代志怪 · 单元剧 · 侠义 · 悬疑
> **文风**：祭酒《地煞七十二变》风格（半文半白、轻快江湖气）
> **结构**：纯古代线，玉佩触发穿越 → 昏迷传送 → 新案件
> **金手指**：一枚来历不明的玉佩
> **终点**：玉佩刻痕亮满 → 赵匡胤魂魄投胎
> **设计原则**：每一个案件都是一个完整的故事，案件之间用"昏迷传送"衔接

---

## 一、核心概念

```
一个普通上班族，某天夜里在出租屋里打游戏。

一局结束。屏幕跳出"再接再厉"。
他低头去拿水杯——

水杯旁多了一枚玉佩。

白色。温润。形制古朴。
不是他买的。不是别人送的。
就静静躺在那儿，好像在等人。

他拿起来。指尖一凉。
还没来得及多想——

眼前一黑。

再醒来时——
身下是稻草。头顶是漏雨的瓦。
门外是唐末乱世的夜风。

玉佩还在他手心里。

每当他解决一个地方的妖鬼之祸，
玉佩便闪一道光，他随即陷入昏睡。
再醒来，已在另一个时间、另一个地点。
玉佩的刻痕多亮一道，
而他多了一项新的能力。

他不知道这枚玉佩从哪来。
不知道它为什么要带着他走遍这乱世。
他只知道——

每次醒来，都有人需要他。
那些他从未见过的人，
那些快被碾碎在这乱世里的人。

而玉佩上需要亮满的刻痕——
是赵匡胤。
一个困在玉中千年的魂魄。
刻痕亮满之日，便是他投胎之时。
```

**一句话摘要**：一个上班族被一枚来历不明的玉佩带到唐末乱世，每到一个地方帮一群人、斩一个妖鬼，就被送往下一个地方——回不去，停不下，路的尽头是一个困了千年的魂魄在等投胎。

## 二、核心机制：穿越与昏迷传送

### 2.1 触发规则

```
【首次穿越】
  打游戏到深夜
    ↓
  发现水杯旁凭空多出一枚玉佩
    ↓
  拿起玉佩，指尖一凉
    ↓
  眼前一黑
    ↓
  醒来 → 唐末乱世，面对第一个妖鬼

【此后传送】
  案件解决
    ↓
  玉佩闪一道光
    ↓
  主角陷入不可抗拒的昏睡（时长不定）
    ↓
  醒来 → 已在新地点、新时间
```

### 2.2 传送特性

| 特性 | 设计 | 叙事效果 |
|------|------|---------|
| **不可控** | 主角无法选择何时离开、去哪里 | 每次都是"突然拔走"，制造无奈感 |
| **时间跳跃** | 可能跳过数周、数月，甚至数年 | 前一个案件的后果主角可能永远看不到 |
| **空间不限** | 从江南到北境，从市井到荒村 | 世界观自动展开，无需旅行章节 |
| **保留记忆与能力** | 醒来时记得一切，解锁的能力都在 | 成长是真实的，但人际关系不断重置 |
| **随身物品** | 玉佩 + 穿越时身上穿的/兜里揣的 | 就一套现代衣服和一部没信号的手机 |
| **回不去** | 没有回现代的路 | 他不是来旅游的——他被困住了 |

### 2.3 叙事节奏

```
┌──────────────────────────────────────────────────┐
│                                                  │
│   【案1】汴州      【案2】洛阳      【案3】太原  │
│   调查→解谜→战斗   调查→解谜→战斗   调查→解谜→战斗│
│   →解锁能力       →解锁能力       →解锁能力     │
│       ↓               ↓               ↓        │
│     昏迷            昏迷            昏迷        │
│       ↓               ↓               ↓        │
│   【醒来】新地点    【醒来】新地点    【醒来】     │
│                                                  │
│   没有"回家"，没有"休整"——                       │
│   只有下一个案子。                                │
│   地煞还能回现代吃泡面，                          │
│   李归舟连这个退路都没有。                        │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 2.4 "被推着走"的无力感（核心情感）

- 一个现代上班族，被扔进唐末乱世，回不去。
- 每次刚在一个地方建立一点关系，就被强制送走。
- 他不知道下一站去哪，不知道这条路有没有尽头。
- 他只知道玉佩上需要亮满的刻痕还有很多道。
- **这本身就是人物的弧光**：一个被迫当侠客的人，从抗拒到接受，从"为什么不让我留下"到"好吧，下一个是谁需要我"。

## 三、主角设定

### 3.1 李归舟

```
【现代身份】
  - 姓名：李归舟
  - 年龄：28岁
  - 职业：办公室职员（财务/行政类），在一家小公司做报表
  - 学历：普通二本，会计专业，跟历史、武术、玄学完全不沾边
  - 家庭：父母在老家，独居在某二线城市出租屋
  - 生活状态：
    早上挤地铁 → 做报表 → 开会 → 午饭外卖 → 做报表 →
    加班 → 挤地铁回家 → 打游戏 → 睡觉
    日复一日，没有惊喜，也没有灾难。
    就是一个"路人甲"。

【性格——本质上的侠肝义胆，但从未激活】
  李归舟骨子里是一个有底线、有侠气的人。
  但这颗"侠胆"从来没被真正激活过。

  他的"勇敢"只存在于想象中：
  - 地铁上看到有人偷手机，他想管，但看到对方有同伙，没敢动。
  - 同事被甩锅，他想帮腔，但领导一眼扫过来，嘴闭上了。
  - 刷到社会新闻评论区义愤填膺，打字打到一半又删了。
    "算了，关我什么事。"

  他不是懦弱。
  他只是从来没被逼到非出手不可的那一步。
  他的生活太平了——平到不需要勇敢。

  他的本质：
  ✓ 理性：做事有逻辑，先算清楚再动手
  ✓ 有底线：再怕也不会做亏心事
  ✓ 看到不公会难受，憋在心里
  ✓ 帮了人会心虚地摆手"没事没事"
  ✓ 别人动感情时他会先把脸别过去
  ✓ 吃软不吃硬，越被威胁越不低头
  ✓ 对老人、孩子、弱者有本能的保护欲
  ✗ 但以上全都没真正派上过用场——直到那枚玉佩出现。

【第一次穿越的遭遇】
  不是"我穿越了，太酷了"。
  而是——

  先是懵。然后慌。然后找手机。手机没信号。
  拼命掐自己看是不是做梦。
  试图睡觉——"睡醒就回去了"。
  睡了一夜，醒来还是稻草和漏瓦。
  恐惧——"公司下周的报表还没交。"

  走出破屋。汴州郊外。秋雨。
  老农路过，说今上是朱温。
  "嗡"的一下。唐末。五代。人吃人的年头。
  他终于肯承认了——这不是梦。回不去了。

  然后他遇到了一个人。
  一个老道士。青布道袍，打满补丁。牵一匹瘦马。
  这人道号玄义。云游四方，无门无派。

  从这天起，玄义老道带着他。
  教他妖分三等。教他鬼怕火和铜器。
  教他道术是手段，剑才是道理。
  教他——"看到不平事就得管"。

  他做的第一件自己都没想到的事：
  一个做报表的上班族，每天劈柴、画符、背口诀。
  嘴上骂骂咧咧："我一个会计，你让我学这？"
  手上没停过。

  直到玄义老道倒下那天。
  他穿上那件道袍。拿起那柄剑。
  剑上刻着"苍生太平"。

【内心驱动】
  - 表层：想回去。星期一还要上班。
  - 中层：回不去的每一天里，玉佩不让他停。
  - 深层：每次看到那些快被碾碎的人，手还是比脑子快。
  - 终极：赵匡胤的魂魄困在玉里。他不知道还要斩多少妖鬼才能让他投胎。
    但他知道被困住是什么感觉——所以他想快一点。

【师承·玄义老道】
  在汴州郊外，李归舟遇到了改变他命运的人——
  一个游方道士。道号玄义。云游四方，无门无派。

  玄义老道五十来岁，灰白头发，青布道袍打了无数补丁。
  牵一匹瘦马，马背上挂一柄剑、一个葫芦。

  相遇并不传奇——主角跟上去，只是为了讨碗粥、找个活路。
  玄义老道留他，是因为发现他能"见鬼"。
  一个开眼的人，在乱世里能当一双好用的眼睛。
  各取所需罢了。

  但处着处着就不一样了。
  玄义老道教了他一切：
  - 妖分三等：怨、煞、孽
  - 鬼怕三样：火、铜器、道诀
  - "道术是手段，剑才是道理"
  - "别学那些假道士骗香火钱"
  - "看到不平事就得管。管得了管不了先不管——得管"

  白天赶路教学。晚上破庙劈柴练剑。
  玄义老道有时喝酒，话多一点。说他年轻时也有个师父，道号冲微，死在蛟嘴里。
  "妖这种东西，除掉一只还有一只。除不完的。"
  李归舟问：那为什么还除？
  玄义老道没答。把酒葫芦递给他。

  汴州饿妖之战。玄义老道倒下了。
  火墙抽干了他最后一口气。

  临终前，他把剑塞进李归舟手里。
  剑柄上刻着四个字：苍生太平。

【继承·衣钵】
  李归舟脱掉格子衬衫。
  换上玄义老道那件打满补丁的青布道袍。袖子长了一截。
  拿起刻着"苍生太平"的剑。
  牵过那匹瘦马。

  从这一天起，他不再是一个穿越来的会计。
  他是一个道士。
  道术玄义老道教的。剑玄义老道给的。马玄义老道留下的。
  连"看到不平事就得管"这句话——也是玄义老道说了一路的。

  玄义老道埋在李家庄的残墙下。
  没有碑。剑尖在墙上刻了两个字：玄义。
  够了。

  此后每一个案件，李归舟都穿着那件青布道袍。
  剑挂在马鞍旁。剑柄上的"苍生太平"磨掉了半截漆。
  他端碗喝粥的时候，有时候会想起玄义老道——
  想起他第一次喝玄义老道的酒，呛得眼泪出来。
  玄义老道在旁边笑。

【成长弧】
  案一前半：学徒期（跟着师父学，被动接受）
  案一后半：断裂与继承（师父牺牲，穿上道袍拿起剑）
  案2-3：独行入门（独自面对，磕磕绊绊，但不再想"回家"）
  案4-6：接受阶段（开始习惯、学会基本生存、"我就是一个道士了"）
  案7-10：主动阶段（不再逃避、会主动打听"这里有什么不对劲"）
  案11+： 侠义成型（不需要别人求他，他自己会站出来）
```

### 3.2 玉佩（金手指）

```
【外观】
  - 一枚白色玉佩，圆形，直径约三厘米
  - 玉质温润，形制古朴，无任何文字
  - 表面有多道浅淡的刻痕——初始时全是暗的
  - 每消灭一个妖鬼，一道刻痕亮起微光
  - 亮满之日，便是赵匡胤投胎之时

【来源】
  - 打游戏那晚凭空出现在水杯旁
  - 不是任何人的礼物，不是祖传遗物
  - 来历成谜——这是全书的终极悬念之一
  - 玉佩从不解释自己，从不"说话"
  - 唯一的信息：消灭妖鬼时，主角心里会浮起一个念头——
    那是玉佩的"意志"，不是语言，更像直觉

【穿越机制】
  - 首次：拿起玉佩 → 指尖一凉 → 眼前一黑 → 醒来已在古代
  - 此后：斩妖完成 → 玉佩闪光 → 昏睡 → 醒来新地点
  - 每次传送后，玉佩会引导他找到下一个妖鬼（模糊的直觉）
  - 玉佩在任何情况下都不会丢失——丢了也会自动回到身上

【能力体系】
  ① 见鬼（基础能力，第一次穿越后自动获得）
     - 能看到鬼魂和妖怪
     - 能触碰鬼物

  ② 斩妖术法（核心成长）
     - 每消灭一个妖鬼 → 玉佩闪一道光 → 解锁一个新术法
     - 术法可以用在后续的战斗/调查中
     - 解锁的术法取决于所斩妖鬼的特性
     - 术法类型示例：
       通幽（见鬼/触鬼）、驱神（役使小灵体）、
       借风（加速/短距飞行）、布雾（遮蔽视线）、
       禁水（水下呼吸）、斩妖（破甲攻击）、
       定身（短暂束缚）……

  ③ 妖气感知（被动能力，中期自然形成）
     - 附近有妖鬼 → 玉佩微微发温
     - 强大的妖鬼在附近 → 玉佩明显发烫
     - 越强大的越烫——主角靠这个判断能不能打得过

  ④ 执念感知（高级能力，中期解锁）
     - 能看到鬼魂/妖怪生前的最后画面
     - 了解它们为何在此
     - 不是用来消灭，是用来理解

【限制】
  - 玉佩不能预知未来
  - 玉佩只对妖鬼有反应——纯"人祸"它不管
  - 每次术法使用消耗体力/精力，过度使用会脱力晕倒
  - 穿越中如果死了——不知道。主角不敢试。
  - 玉佩从不解释自己。来历、目的、为什么选中他——全不知道。
  - 唯一的确定性：亮满刻痕 = 赵匡胤投胎。这是玉佩给主角的唯一"承诺"。

【关键谜团（留给读者）】
  - 玉佩是谁做的？
  - 赵匡胤的魂魄为什么困在里面？
  - 为什么选中一个普通上班族？
  - 刻痕亮满之后，主角会怎样？
  → 这些问题全书逐步揭示，但不要一次给完。
```

## 四、案件（单元剧）设计模板

### 4.1 每个案件的标准结构

```
【第一章：落地】
  - 主角从昏迷中醒来
  - 环境描写：在哪里？什么时间？什么季节？
  - 第一个感官细节：闻到了什么？听到了什么？
  - 玉佩微微发温（提示附近有妖鬼）
  - 见到第一个当地人物

【第二章：调查】
  - 收集信息：询问当地人、走访现场
  - 发现异常：什么不对劲？
  - 信息渐进：不一次性给出全部设定
  - 主角的腹诽/吐槽（节奏调节）

【第三章：深入】
  - 第一次遭遇妖怪/鬼物
  - 不是正面战斗，是试探/被袭击/目击
  - 发现案件的关键线索
  - 妖怪的"因果"开始浮现

【第四章：真相】
  - 还原妖怪的来历
  - 必定有"人间因果"：
    贪/嗔/痴/怨/执念/冤屈 → 死后化为妖鬼
  - 不是单纯的"坏"，而是"惨"
  - 关键反转点（预期颠覆）

【第五章：解决】
  - 战斗/超度/谈判/化解（不一定是暴力）
  - 用能力还是用脑子？最好是"巧胜"
  - 玉佩在掌心发烫 → 妖鬼消散 → 玉佩闪一道光
  - 解锁新能力（主角能感觉到一个新的"东西"进入了身体）
  - 余波：当地的后续
  - 主角感到昏睡袭来
  - 最后一句话/画面（留给读者回味）
```

### 4.2 案件多样性要求

| 类型 | 占比 | 示例 |
|------|------|------|
| **恐怖悬疑** | 40% | 鬼婴、僵尸、鬼市、尸宴 |
| **温情悲剧** | 25% | 白狐拜月、士兵等祭祀、老人守神像 |
| **江湖恩怨** | 20% | 活人术士作恶、教派冲突、官匪勾结 |
| **黑色幽默** | 15% | 鬼屋室友、怂狐狸、假道士骗钱 |

**换口味原则**：连续两个案件不能同类型。

### 4.3 案件必须遵守的规则

- ✅ 每个妖怪/鬼物必须有来历（因果闭环）
- ✅ 来历必须是"人间事"（贪婪、冤屈、执念、爱与恨）
- ✅ 解法不一定是暴力（超度、谈判、满足遗愿、揭露真相）
- ✅ 主角每次解锁一个新能力
- ✅ 每个地方有一个"留下来会很好"的瞬间（然后被传送走）
- ❌ 不能有纯粹的恶（妖怪也有人性面/苦衷）
- ❌ 不能重复同类型的因果（不能连续两个都是冤案）

## 五、案件示例（前五个）

### 案一：汴州·饿妖

```
【地点】汴州城外三十里，李家庄废墟
【时间】907年深秋
【类型】恐怖悬疑 + 师承 + 硬战

【异常现象】
  李家庄四十七口人，一夜之间全没了。
  锅还在灶上。饭还在碗里。人没了。

  地上有骨头。啃过的。
  不是鸡骨鱼骨。

  此后附近村子开始丢人。
  先是乞丐。然后是赶夜路的。然后是白天在地头的。

  有人见过——七八个穿破军服的"人"，
  在荒地里走。走得慢。嘴在动。一直在动。
  看见活人就追。追上就围。然后是一阵啃咬。

【落地·初遇师父】
  主角在汴州城外一栋破屋里醒来。身下烂草席，头顶秋雨。
  他先摸口袋——手机还在，电量78%，信号一格没有。
  "操。"
  低头看了看自己：格子衬衫，牛仔裤，运动鞋。
  抬头看了看周围：漏雨的瓦，土坯的墙，门外的驴粪蛋。
  "这什么地方。"

  一个老农路过，看见他，吓得往后退了一步。"后生，你咋穿这衣裳？"
  主角站起来。问：这是哪儿？
  老农说：汴州城外。今上是朱温。
  主角脑子里"嗡"了一下。
  唐末。五代。人吃人的年头。
  穿越了。不是做梦。他掐过大腿了。青了。

  老农摇摇头走了。主角蹲在破屋门口，不知道该往哪走。
  没有钱。没有吃的。没有身份。身上就一件格子衬衫和一部没信号的手机。
  手机还快没电了。

  然后他看见一个人。

  一个老道士。五十来岁，灰白头发潦草扎着，青布道袍打了好几个补丁。
  牵一匹瘦马。马背上挂着一柄剑，一个葫芦。

  老道从路上过。看见破屋里蹲着个人，瞥了一眼。
  格子衬衫。牛仔裤。运动鞋。
  他多看了两眼——这穿着不像是中原任何一个地方的服饰。
  但也没停脚。

  主角站起来。"道长。"
  老道回头。
  "这附近有村子没？"
  老道上下打量了他一遍。目光扫过那双运动鞋——鞋底的花纹他没见过的。
  "往前走三里地。有村。"
  说完继续走。瘦马跟在后面，蹄子在泥里一深一浅。

  主角跟了上去。
  不是被收留。是他自己跟的。
  一个上班族在唐末荒野里，除了跟着一个看起来还像个好人的老道士，还能跟谁？
  老道没赶他。也没叫他跟上。就是走他的路。

  到了村子。老道进一家农户讨水。农户给了两碗粥。
  一碗给老道。一碗空着。
  老道看了看门口——主角站在那儿，不好意思进来，肚子叫了一声。
  "进来。"
  主角进去。端起粥。一口灌下去。米粒数得清。
  老道问："叫什么？"
  "李归舟。"
  "李归舟。"老道咂摸了一下，"归舟归舟，你归哪儿去？"
  主角答不上来。
  老道又看了他一眼。目光这次落在他脖子上的玉佩绳。
  "那玉，哪来的？"
  "……不知道。捡的。"
  老道没再问。喝完粥，搁下碗，起身。

  主角又跟上了。
  老道没回头。只是说了一句："老道可是要往汴州去的。"
  主角说："我也去汴州。"
  老道没声音。马的响鼻替他答了。

【开眼——师徒的起点】
  傍晚路过一片老坟地。
  主角忽然站住了。
  他看见坟头上蹲着个东西。灰白色的。像一团烟。有个人脸的轮廓。

  老道已经走出去好几步。察觉身后脚步停了。回头。
  "怎么不走？"
  "那儿……有个人。"
  老道顺着他的目光看过去。坟头。空空如也。
  他又看回来。盯着主角的眼睛。
  "你看见了？"
  "……看见了。灰白色的。像个男的。"
  老道沉默了一会儿。

  然后走过去。从怀里摸出一张黄纸符。往坟头上一贴。
  灰白烟团缩了一下。钻进土里。
  "没事了。走吧。"

  继续赶路。走了一刻钟。老道忽然开口：
  "我这辈子见过两个天生开眼的。一个是我师父。一个是……"
  他停了一下。
  "寻常人看不见的东西你能看见。不一定是好事。也不一定是坏事。"
  又走了几步。
  "你跟着我走。能帮上些忙。我教你几手。"
  主角想了想。"学费多少？"
  老道看了他一眼。那眼神像在看一个憨货。
  "帮我喂马。劈柴。打水。多双眼睛帮我盯路。"
  主角想了三秒钟。点头。
  他一个做报表的，现在要给一个老道士劈柴喂马换饭吃。
  但总比蹲在破屋里饿死强。

【师承——道号玄义】
  老道道号玄义。云游四方，无门无派。
  主角叫他玄义道长。有时候直接叫老道。老道也不恼。

  白天赶路。玄义老道边走边教：
  · 妖分三等——怨（新死不久）、煞（百年积累）、孽（吃过人的）
  · 鬼怕三样东西——火、铜器、道诀
  · "这世道乱的根子在人心。妖只是长出来的果子。"
  · "道术是手段。手里的剑才是道理。"
  · "别学那些假道士骗香火钱。丢人。"

  晚上宿在破庙。玄义老道逼他练剑。
  木剑劈柴。一千下。手腕肿了也得劈。
  "现在疼，总比遇妖的时候脖子疼强。"

  主角——一个做报表的——现在每天劈柴、喂马、画符、背口诀。
  嘴上骂骂咧咧："我一个会计，你让我学这？"
  手上没停过。

  这样过了大半个月。主角瘦了，黑了，手上全是茧。
  腿不抖了。木剑劈柴，手也不抖了。

  玄义老道有时候喝了酒，话多一点。
  说他年轻时也有个师父。道号冲微。死在一条蛟嘴里。
  "妖这种东西，除掉一只还有一只。除不完的。"
  主角问：那为什么还除？
  玄义老道没答。只是把酒葫芦递给他。
  主角灌了一口。呛得眼泪出来。
  玄义老道笑了。
  "好酒。可惜就这么一葫芦。"
  主角不知道他说的"好酒"是真的还是假的。他只知道自己被呛出的眼泪沾了满脸灰。
  但他没擦。
  玄义老道也没笑他。

【调查】
  两个人到了汴州城外。城门口贴着悬赏：李家庄妖患，能除者赏银二十两。
  玄义老道盯着悬赏看了一会儿。
  "二十两。够买一匹好马。"看了看自己的瘦马，"算了，这匹也还行。"

  走访三个村子。一老一少搭档——
  玄义老道问话。主角在旁边听。偶尔记两笔。
  · 怪物只在黄昏和夜晚出没
  · 怕火。有村民用火把逼退过一次
  · 领头的是个大胡子，穿破军服，走路拖一条腿
  · 李家庄为中心，圈子在扩大

  两人去了李家庄。
  四十七间空屋子。灶台上锅还在，锈了。饭碗长了绿毛。

  地上有骨头。
  玄义老道蹲下去。拿起一根。看了看。
  "人的。指骨。牙印齐整。"
  又拿起一根。"肋骨。人的。咬得狠。"
  他站起来。脸色不太好。
  主角站在旁边，胃在翻。但没吐——玄义老道教过他，见妖之前先别自己吓自己。

  玄义老道把骨头放回原处。
  "吃人的妖。不是新死的。年头不短。"
  他环顾四周。"黄巢的溃兵。二十三年前的孽。"

  当夜，找到了那个瘸腿老兵。老兵给人补鞋为生，一听"李家庄"三个字就沉默了。
  "那是赵都头。"老兵说，"黄巢的兵。"
  
  二十三年前——884年冬天。黄巢兵败。赵都头带七个人逃到汴州。没粮。
  先吃死人。后吃活人。吃完了一个村子。
  最后他们互相吃。赵都头活到最后。
  "你知道他最后吃的啥？"
  老兵指了指自己嘴——
  "自己的舌头。饿疯了。嚼自己嚼到断气。"

  死后变成妖。
  不是冤魂。不是怨灵。是"吃"这个行为刻进了魂里。
  他们不知道自己吃什么。只知道饿。永远饿。

  玄义老道放下碗。沉默了很久。
  然后说了一句话——不是对主角说，倒像在对自己说：
  "二十三年的老妖。七个。不太好办。"

【危机·师父倒下】
  深入那天晚上。荒野上。七个影子出现了。

  玄义老道拔剑。剑出鞘的声音很脆。像折断一根冬天的树枝。
  "归舟。往后退。"
  语气跟平时教他练剑一模一样。跟叫他从柴堆上下来一模一样。

  然后一个人挡在前面。
  一剑劈开一个妖。剑光在残墙之间闪了三次。逼退两个。
  这种身手——主角从来没在玄义老道身上见过。

  但老了就是老了。第七剑的时候，赵都头从侧面撞上来——
  玄义老道飞了出去。后背砸在残墙上。墙塌了半边。

  主角冲过去。
  玄义老道嘴角有血。剑还在手里。
  "木剑练了一段日子了。试试真剑。"
  把剑塞进他手里。剑柄还是热的。

  然后玄义老道撑着站起来。双手掐诀——
  一道火墙从地上蹿起来。火光映在他脸上，蜡黄蜡黄。
  七个妖被隔在火墙另一侧。嘴还在动。咀嚼声隔着火传过来。

  "火只能撑一盏茶。"
  玄义老道从怀里摸出一个小布袋。灯油。引火用的。一直备着。
  "你知道怎么做。"

  主角攥着剑。手在抖。
  "老道，你呢？"
  玄义老道没答。靠在残墙上。闭了眼。

  主角转身。
  七双空洞的眼睛——在火墙另一边等着。

【解决】
  主角用玄义老道教过的法子。

  荒野上。枯草和破布浇上灯油，围成一个直径三丈的圈。
  圈中间：一具山羊尸体。跟村民换的。
  道诀加持过的火——比村里灯油猛十倍。

  黄昏。他蹲在上风口的土坡后面。
  七个影子出现了。一步一步。拖拖拉拉。嘴都在动。
  围住山羊。开始啃。

  主角冲出去。打火机擦着枯草——
  火苗蹿起来。风助火势，呼啦一下围成火墙。
  不是灯油的火。是道火。青蓝色。

  七个东西被火困在圈里。没有惨叫。只有噼啪声。夹杂着——
  咀嚼声。火已经烧到身上了，还在嚼。

  最后一个站着的是赵都头。一条腿烧没了，半边身子在火里。
  他忽然抬起头。嘴停了。
  看了主角一眼。
  没有怨恨。没有感激。什么也没有。
  然后散了。像火里的灰。

  主角没有看灰。
  转身往回跑。

【继承·衣钵】
  残墙下。玄义老道还靠在原处。
  火墙灭了。他的脸色跟墙一样灰。

  主角蹲下去。手搭上玄义老道的脉。
  没有了。

  火墙抽干了他最后的气。

  主角坐了很久。久到星光出来。久到瘦马走过来，用鼻子碰了碰玄义老道的袖子。

  然后他站起来。
  脱掉格子衬衫。换上玄义老道那件打满补丁的青布道袍。
  不是很合身。袖子长了一截。

  他拿起玄义老道的剑。
  剑柄上刻着四个字——
  苍生太平。

  他把玄义老道埋在残墙下。
  没有碑。就用剑尖在墙上刻了两个字：玄义。

  牵那匹瘦马。马看了他一眼。没躲。
  "走吧。"
  不知道是对马说，还是对自己说。
  "替你老道把路走完。"

【反转】
  火灭后的灰烬里，一块铜牌烧不坏。
  上面八个字：
  "左神策军第十营 赵二郎"

  神策军。大唐禁军。
  黄巢破长安时，这七个人跟着队伍跑了。
  从禁军变成流寇，从流寇变成饿鬼，从饿鬼变成妖。
  最后连舌头都嚼了。

  主角捡起铜牌。看了看。
  然后把它埋进玄义老道的坟旁。
  两个死人——一个吃了二十三年人的人，一个救了一辈子人的道。

  二十三年的妖，临散之前嘴停了一瞬。
  算不算听到"归队"了？
  主角不知道。
  但他想——玄义老道应该是听到了。

【解锁能力】通幽（见鬼、触鬼——从模糊感知转为清晰可视化）
【玉佩闪光】第一道刻痕亮起。

【离开画面】
  他骑上瘦马。道袍被秋风掀起一角。
  剑挂在马鞍旁。沉甸甸的。

  玄义老道说的话还在耳朵里转：
  "道术是手段，剑才是道理。"
  "别学那些假道士骗香火钱。"
  "看到不平事就得管。管得了管不了先不管——得管。"

  他还没走出三里地。
  困意砸下来。
  从马背上摔下去。

  最后看见的——那匹瘦马站在身边，没跑。低头啃了一口枯草，打了个响鼻。
  然后天旋地转。

  再醒来。
  身下冻土。头上北风。远处磷火。
  衣领被什么扯了一下。
  瘦马叼着他的道袍袖子，往旁边拽。

  剑还在。道袍还在。马还在。
  有人在哭。
```

### 案二：洛阳·冻土饿鬼

```
【地点】洛阳城外，白马寺附近的乱葬岗
【时间】908年隆冬（跳了几个月）
【类型】温情悲剧

【异常现象】
  乱葬岗晚上有火光亮起。
  附近村民说，那是"饿鬼在烤火"。
  但去过的人回来都说，那火不热——是冷的。

【调查过程】
  主角到洛阳 → 打听 →
  夜晚摸到乱葬岗 →

  看到几百个鬼魂围成圈。
  中间一堆磷火，蓝幽幽的，冰凉的。
  它们不是在烤火。
  它们在"等饭"。
  等了三年了。

【真相】
  三年前一场大战，梁晋双方各死数千人。
  尸体被拖到这里草草掩埋。
  没有人来收尸，没有人来祭祀。
  饿鬼们的执念：不是报仇，是"等有人来给我们烧纸"。

【解法】
  主角到附近村子，说服村民组织了一次集体祭祀。
  当纸钱在风中燃起，磷火忽然变暖了。
  饿鬼们的身体慢慢变淡。
  一个年轻士兵鬼问："我娘还活着吗？"
  村民中有个老太太，颤颤巍巍走过去——
  "是我儿。是我儿的魂。"
  士兵鬼笑了笑，散了。

【反转】
  这些饿鬼三年没有害过任何一个村民。
  它们只是冷，只是在等。
  "打仗的时候没人问我们冷不冷，
   死了还是在等。"

【解锁能力】驱神（役使小灵体，可以让小鬼帮忙侦察）
【玉佩闪光】第二道刻痕亮起。

【离开画面】
  墓地上新立的木碑。
  老太太把一碗热粥放在碑前。
  主角靠着碑坐下，忽然很困。
  醒来时——
  听到的不是洛阳的北风，
  而是水声。涛涛江水，拍打船帮。
```

### 案三：钱塘·老蛟

```
【地点】钱塘江边，水神庙
【时间】909年夏（跳了大半年）
【类型】社会隐喻 + 奇幻

【异常现象】
  钱塘江上连续翻船。
  渔夫说江里有一条蛟，已经闹了大半年。
  更古怪的是——水神庙的香火，断了。

【调查过程】
  主角到达 → 观察 →
  发现水神庙里神像被搬走了，取而代之的是一尊财神。
  原来的水神像被丢在江边，半埋在淤泥里。

【真相】
  蛟是水神的坐骑，原来的水神。
  但自从庙里换了财神，香火断了两百年，
  它越来越虚弱，越来越暴躁。
  它不是想伤人。
  它是想让人想起它。
  "我不需要香火，我只是不能让船翻了。
   那是我当了六百年水神唯一的职责。
   我快撑不住了。"

【解法】
  主角不是斩蛟，而是帮它把水神像搬回庙里。
  渔民重新点了香。
  蛟在江中翻身，掀起最后一波浪——
  浪推着所有渔船，平安靠岸。

【反转】
  蛟不是被邪气污染的受害者。
  是被人遗忘的守护者。
  它伤人的每一次，都是在喊："我还在。"
  但没人听得懂。

【解锁能力】禁水（水下呼吸/水中行动）
【玉佩闪光】第三道刻痕亮起。

【离开画面】
  主角站在江边，看着蛟的脊背没入江心。
  他忽然想到——自己被困在这个时代，
  像一条被遗忘的蛟。
  困意袭来。
  醒来时——
  听到的不是钱塘江的涛声，
  而是风沙。北地的风沙，夹杂着马嘶。
```

### 案四：太原·两棵杏树

```
【地点】太原城外，废弃的村庄
【时间】910年春（跳了小半年）
【类型】民间异闻 + 犯罪悬疑

【异常现象】
  村口有两棵百年杏树，今年春天莫名其妙开了红花。
  不是粉色桃花那种——是血红色。
  而且树下晚上有人影，一男一女。
  走近就没了。

【调查过程】
  主角到达 → 询问 →
  村民说这树下三十年前埋了一对夫妻。
  男人是村里的猎户，女人是邻村织娘。
  两人在杏树下约会被发现，被族人浸了猪笼。
  此后杏树每年开白花。
  唯独今年，开了红花。

【深入调查】
  主角挖开树下——
  没有尸骨。
  只有一口空棺材，棺材里全是手指印。
  有人被活埋了，又爬出来了。
  那么，是谁？

【真相】
  三十年前的"浸猪笼"，实际上不是被发现的，
  是被一个觊觎女人的有钱人举报的。
  事后，有钱人谎称替他们收尸，
  实际上男人被卖给了军队当炮灰，
  女人被关在地窖三十年。
  一个月前，女人死了。
  男人回来了——在战场上断了一条腿，成了残废。
  他回来第一件事，就是在杏树下挖出了女人的尸体，
  重新埋好。
  然后自缢在树上。

  树下的"人影"，是他们在杏花里重逢。
  红花，是血的执念。

【解法】
  主角找到当年那个有钱人——
  他已经是个七十多岁的老头，儿孙满堂。
  主角什么都没做。
  只是当着全村人的面，把他当年做过的事，
  一条一条念了出来。
  第二天，老头自己去了杏树下。
  第三天，杏花变回了白色。

【反转】
  男人三十年没死，是因为他答应过女人：
  "我会回来，在杏花开的时候。"
  他回来了。
  杏花果然开了——只是红色的。

【解锁能力】斩妖（对妖鬼的破甲攻击）
【玉佩闪光】第四道刻痕亮起。

【离开画面】
  主角再次经过杏树。
  树下有个年轻猎户和一个织娘并肩坐着，
  在等他经过——然后笑了笑，散了。
  困意袭来。
  醒来时——
  听到的不是北地的风沙，
  而是雨声。密密麻麻落在瓦上的雨。
```

### 案五：燕云·守碑人

```
【地点】燕云十六州边境，一座废弃的烽火台
【时间】912年晚秋（跳了两年多）
【类型】人祸 > 鬼祸

【异常现象】
  边境线上有一座孤零零的烽火台，已经废弃了二十年。
  但每天晚上——
  台上都会点亮烽火。
  上去看，空无一人。

【调查过程】
  主角到达 → 爬山 →
  烽火台里有一个老人。
  不是鬼。是人。
  老人说：他二十年前是这里的守卒，
  敌军来袭时，他点燃烽火报信。
  烽火烧了一整夜。
  援兵没有来。
  他守了二十年，每天晚上继续点。
  因为——
  "万一今晚有敌人呢？"
  "万一今晚援兵来了呢？"

【真相】
  这场仗二十年前就打完了。
  老人知道。
  但他还是每天点。
  "我不是在等敌人，
   我是在等有人来告诉我——
   你的任务完成了。"
  二十年间，没有任何人来说这句话。
  将军忘了这里。朝廷忘了这里。
  只有老人没忘。

【解法】
  主角陪老人点了一夜的烽火。
  天亮时，
  主角说："你的任务完成了。"
  老人看了看他，
  把火把递给他。
  然后把穿了二十年的军服叠好，
  放在台上。
  走下山的时候，
  老人回头看了一次——
  烽火台上，二十年前那个年轻的士兵站在那里，
  朝他敬了一个军礼。
  然后熄了。
  这次是真的熄了。

【反转】
  老人是人，不是鬼。
  但二十年的等待本身，已经成为一种执念。
  比鬼更顽固，比妖更执着。

【解锁能力】借风（短距离快速移动/加速）
【玉佩闪光】第五道刻痕亮起。

【离开画面】
  主角在烽火台下睡着了。
  醒来时——
  听到的不是燕云的朔风，
  而是……沉默。
  绝对的、无边的沉默。
  他躺在一片废墟中，四周是断壁残垣。
  一个孩子蹲在他身边，说：
  "叔叔，你睡了好久。我们村昨天晚上又死了一个人。"
```

## 六、主线暗线：玉佩之谜 × 赵匡胤投胎

### 6.1 明线：消灭妖鬼，点亮玉佩

每个案件都是独立的斩妖除魔故事。玉佩上的刻痕一道一道亮起。这是读者看得见的主线进度条。

### 6.2 暗线：赵匡胤的魂魄

从案三左右开始，可以在穿越间歇出现赵匡胤的意识碎片：

| 阶段 | 线索 | 表现形式 |
|------|------|---------|
| 案1-2 | 玉佩只发出任务信号，无额外信息 | 主角只知道要消灭妖鬼 |
| 案3 | 第一次隐约感知：玉佩里有一个意识 | 某次斩妖后，感觉到"有人在看" |
| 案4-6 | 第一次见到碎片画面——一个穿甲胄的人影 | 昏睡传送时一闪而过的残影 |
| 案6 | 在昏睡中短暂对话（一句话） | 那个声音说："还差很多" |
| 案7-10 | 对话渐多。赵匡胤偶尔在斩妖前给提示 | "这个妖怕火" |
| 案10 | 一次关键的对话——赵匡胤说出自己是谁 | "吾名赵匡胤。困于此玉，已千年。" |
| 案11-15 | 赵匡胤的执念浮现——不是皇位，不是江山 | "我想知道大宋后来如何了" |
| 案15+ | 最后几道刻痕。赵匡胤越来越虚弱。 | "不必勉强。千年都等了，不差这几日。" |

**赵匡胤的设定**：
- 他不是威严的帝王，是一个困在玉里千年的魂魄
- 他记不清关于自己的一切了——只记得一些碎片
- 他最想知道的是：大宋后来怎样了？
- 但李归舟不知该怎么开口——大宋？三百年。最后被蒙古灭了。
  "你投胎之后，自己去看吧。"
  这是他唯一能给的回答。

### 6.3 暗线：玉佩的来历

玉佩的来历在前期完全不给答案。线索极其克制：

| 阶段 | 线索 |
|------|------|
| 案1-5 | 零线索。主角完全不知道玉佩的来历 |
| 案6-10 | 偶尔在穿越间隙见到其他"戴过这枚玉佩的人"——都是普通人，都死了。（残影，一闪而过） |
| 案11-15 | 发现玉佩不只一枚。有另一枚——在一个活人手里。 |
| 案16+ | 逐渐揭示：玉佩是某个"守玉人"传承体系的信物。主角是当代守玉人。 |

**最终揭晓方向（不锁死）**：
玉佩是唐代一位高人以陨铁与魂魄石所铸，共制七枚，每一枚困着一个"不该死在那个时候"的人的魂魄。赵匡胤是其中之一。守玉人的使命是用斩妖积攒的灵力，把魂魄重新送回人间。

> **注意**：暗线极其克制。每3-5个案才出现一次线索。
> 案件本身永远是第一位的。暗线是盐，不是主菜。

## 七、写作风格（参照《地煞七十二变》祭酒文风）

### 7.1 语言规则

- **半文半白**：用"甫一""悚然""径直""好似""腹诽""嗤笑""面面相觑"但不说"余""汝""窃以为"
- **短句为主**：动作描写不超过15字/句，恐怖场景用1-6字断行制造节奏
- **开篇即钩**：第一句定调（4-12字）
- **细节恐怖**：不写"恐怖""可怕"，只写具体视觉/嗅觉/触觉细节
- **反差幽默**：每3-5段紧张后插入一句腹诽或吐槽，但不破坏氛围
- **对白口语化**：角色说话带烟火气
- **因果闭环**：每个妖怪/鬼物有来历，背后是人间因果
- **悲剧用白描**：不煽情，陈述事实
- **主角的现代思维**：李归舟是上班族穿越过去，他看古代的一切都带着现代的吐槽视角——这是幽默的最大来源

### 7.2 叙事节奏

每个案件控制在3-5万字（约8-15章）。
全书初期规划15-20个案件（对应15-20道刻痕）。
每个案件像一集电视剧，可以独立观看，但连续阅读体验更好。
