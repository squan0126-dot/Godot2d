# 世界地理格局

> **状态**：三版设计（唐末乱世重构）  
> **游戏模式**：2D半开放大世界，独立地点可自由前往（非线性）  
> **时代背景**：唐末乱世——藩镇割据、军阀混战、百姓流离、人相食  
> **设计原则**：所有区域同时存在，玩家自由选择前往，不同区域危险度/复杂度不同  
> **开局位置**：江南  
> **核心基调**：整个世界都是乱世废墟，没有"太平盛世"——但废墟中有人在挣扎求生，有人在救治天下

---

## 一、世界总览

### 1.1 时代背景

**大唐已亡，天下大乱。**

- 朝廷名存实亡，皇帝如同傀儡，天天逃命
- 藩镇割据，各地军阀自立为王
- 百姓流离失所，十室九空
- "凤翔地，人肉百文"——人相食已是常态
- 战乱、饥荒、瘟疫三重打击下，人口锐减
- 妖魔鬼怪趁乱横行——怨气冲天，灵气污染，万物异变

### 1.3 政治格局——群雄割据

**朝廷已是空壳**：
- 皇帝被军阀挟持，朝令夕改
- 国师（活了近二百岁的女性修行者）曾是朝廷最后的"底牌"，但她已经**去世**——在故乡造就盛唐幻境后耗尽法力
- 镇抚司名义上还在运作，但人手凋零，各地分部自行其是

**各方势力**：

| 势力 | 原型 | 控制区域 | 特征 |
|------|------|---------|------|
| **沙陀军** | 李克用 | 北境·河东 | 骁勇善战，与朱温死敌，亦正亦邪 |
| **宣武军** | 朱温 | 北境·中原交界 | 野心勃勃，最终篡唐的枭雄 |
| **秦宗权残部** | 秦宗权 | 中原·南部 | **四处吃人**，军队以人肉为粮，与宗教势力勾结 |
| **吴越** | 钱镠 | 江南·东部 | **保境安民**的伟人，在乱世中维持一方太平 |
| **闽地** | 王审知 | 江南·南部 | **开闽圣王**，收容流民，发展经济 |
| **千佛寺** | 虚构 | 江南·山中 | 表面佛门圣地，实则与军阀勾结炼制造化丹 |
| **残存朝廷** | 唐末朝廷 | 中原·某处 | 苟延残喘，国师维持最后的体面 |

### 1.4 经济格局——崩溃与重建

**整体经济已经崩溃**：
- 铜钱贬值，以物易物成为主流
- 粮食比金银更值钱
- 丹药极度稀缺（国师的炼丹体系随朝廷崩溃而瓦解）
- 各地自行铸币，货币混乱

**区域经济差异**：

| 区域 | 经济状态 | 特征 |
|------|---------|------|
| **江南（吴越/闽地）** | 相对最好 | 钱镠/王审知治下有基本秩序，商业勉强运转 |
| **江南（其他）** | 混乱 | 豪强割据，物价飞涨，黑市横行 |
| **中原** | 崩溃 | 战乱反复蹂躏，十室九空，流民遍地 |
| **北境** | 战争经济 | 一切为军事服务，民间物资被征尽 |

## 二、江南（开局区域 / 前中期）

> **总氛围**：天高皇帝远，朝廷管不到这里。有人在废墟中重建秩序（钱镠、王审知），有人在趁乱作恶（豪强、千佛寺）。妖魔横行，但也有人在抵抗。这里是乱世中"最不坏"的地方——所以玩家从这里开始。

### 2.1 开局村落（序章/教程区）

| 项目 | 内容 |
|------|------|
| **所属区域** | 江南·偏远山区 |
| **定位** | 玩家出生点，序章区域 |
| **氛围** | 偏远、贫穷但尚有人烟；远离战乱核心，但妖魔已经开始出没 |
| **势力** | 无（村民自治，无官府管辖） |
| **妖鬼** | 画皮鬼（序章Boss）——黄皮书的第一个目标，从农家吃新郎开始作祟 |
| **NPC** | 村民（委托人）、导师型NPC（引路人/类似刘老道） |
| **经济** | 极度贫穷，以物易物（鸡蛋、粗粮、碎布） |
| **功能** | 学习基础操作、第一场战斗（必败）、了解世界观 |
| **落脚处** | 破庙/借宿民家 |

**设计要点**：
- 主角凭空出现于此（黄皮书选中）
- 遭遇画皮鬼——黄皮书上对应的妖物
- 从村民口中了解世界基本常识："朝廷？早没了。""外面在打仗，别出去。"
- **第一战必败**——画皮鬼太强，主角实力不足
- 画皮鬼逃离村落，消失（后来出现在越州城伪装成权贵）
- 导师NPC指引方向，告知附近有城镇可以落脚

### 2.2 越州城（江南·初期主城）

| 项目 | 内容 |
|------|------|
| **所属区域** | 江南·吴越钱镠治下 |
| **定位** | 玩家前期的主要活动据点 |
| **氛围** | 乱世中难得的"安全岛"——钱镠治下有基本秩序，城墙完好，兵丁巡逻；但城外依然危险 |
| **势力** | 吴越军（钱镠部下）、城中商会、游方道士/僧人 |
| **妖鬼** | 城外有妖（城内相对安全）、偶尔闹鬼 |
| **NPC** | 吴越将领（正派委托人）、商人、导师型道士、旅店老板 |
| **经济** | 江南最稳定的市场——可买卖制符材料、法器；丹药稀缺但偶尔有 |
| **功能** | 接委托赚钱、购买物资、拜师学艺、推进第一章主线 |
| **落脚处** | 旅店（花钱）、城外破庙（免费） |
| **副本入口** | 城外荒山（散妖/僵尸）、附近山贼据点 |

**设计要点**：
- 钱镠治下的城镇——**乱世中的一盏灯**
- 进城需要身份确认（度牒/路引），但不像其他地方那样勒索
- 告示栏有除妖悬赏（正规的，不克扣）
- 能感受到"有人在认真治理"的氛围——与后面看到的其他地方形成对比
- 但城外依然危险——出了城墙就是无人管的荒野
- 从这里的NPC口中了解到：其他地方更乱，有人在吃人（秦宗权），有寺庙在害人（千佛寺）

### 2.3 泉州（江南·闽地主城）

| 项目 | 内容 |
|------|------|
| **所属区域** | 江南·闽王王审知治下 |
| **定位** | 江南另一个"安全岛"，中期活动据点 |
| **氛围** | 王审知收容大量流民，城中人口密集但有序；海商贸易带来外来物资 |
| **势力** | 闽军（王审知部下）、海商、南方宗教势力 |
| **妖鬼** | 海上妖物、山中精怪 |
| **NPC** | 闽军将领、海商（特殊物资来源）、南方术士 |
| **经济** | 海商贸易带来稀有材料；有独特的南方制符材料 |
| **功能** | 特殊物资获取、海上副本、南方支线 |
| **落脚处** | 旅店、商会提供的住所 |

**设计要点**：
- 与越州城形成"双安全岛"格局——玩家在江南有两个可靠的据点
- 王审知的"开闽"政策：收容流民、发展经济、修建道路
- 海商带来中原/北境买不到的特殊材料
- 但闽地也有自己的问题——南方山中精怪多，海上有妖物

### 2.4 千佛寺（江南·宗教副本）

| 项目 | 内容 |
|------|------|
| **所属区域** | 江南·山中 |
| **定位** | **第二章核心副本**——"人比妖可怕"的体现；造化丸的产地（画皮鬼因吃了此处流出的造化丸而变强） |
| **氛围** | 表面：香火鼎盛、佛号声声；实则：与军阀勾结，活人做肉身佛，炼制造化丹 |
| **势力** | 千佛寺高层（与秦宗权残部勾结）、底层僧人（被蒙蔽/被迫） |
| **妖鬼** | 肉身佛（邪术制造）、僧鬼、怨煞 |
| **NPC** | 圆通（线人型）、正派僧人（少数）、受害者家属 |
| **经济** | 寺庙敛财（香油钱、法事费） |
| **功能** | **第二章主线Boss**、揭露造化丹线索、"人比妖可怕"主题 |

**设计要点**：
- **千佛寺与秦宗权残部勾结**：
  - 秦宗权的军队四处吃人，杀人如麻
  - 千佛寺以"超度"为名收容尸体/活人
  - 实则在炼制造化丹——一边杀人吃人，一边炼化人尸
  - 千佛寺得到修为提升，秦宗权残部得到"法术支持"
- 第一章击败画皮鬼后，线索指向此处（画皮鬼吃了从千佛寺流出的造化丸变强）
- 圆通是关键线人——"本是泼皮避兵灾做了和尚"
- Boss：肉身佛/邪僧
- 解决千佛寺 = 第二章结束

### 2.5 荒山/野外（江南·探索区域）

| 项目 | 内容 |
|------|------|
| **所属区域** | 江南·各城镇之间 |
| **定位** | 连接各地点的野外区域，自由探索 |
| **氛围** | 荒凉、危险、无人管辖；曾经的村落已成废墟 |
| **势力** | 无（法外之地）——山贼、散妖、游魂、食人者出没 |
| **妖鬼** | 散妖（白狐、蛇妖等）、僵尸、诈尸、破庙鬼魅 |
| **NPC** | 流民、猎户（可能是食人者）、游方僧道、"黄角式人物"的信众 |
| **经济** | 无商业，但有野生材料可采集（制符材料） |
| **功能** | 探索、采集、随机遭遇、通往其他区域 |
| **落脚处** | 野外营地、破庙 |
| **副本入口** | 山贼巢穴、山中妖窟 |

**设计要点**：
- 赶路时随机遭遇（荒山破庙美人、诈尸、山贼伏击、食人者）
- 有善恶难辨的妖怪（白狐拜月——打还是不打？）
- **"黄角式人物"的踪迹**——在野外能遇到他的信众，穷苦百姓互助的小团体
  - 他们用符水赈灾、收容流民
  - 违法（私自赈灾在乱世中也是"违法"的——因为各方势力都想控制人口）
  - 但他们是善的——这是白莲教的雏形，是穷苦百姓的一份救赎
- 制符材料的主要采集地
- 能遇到从中原/北境逃来的难民——从他们口中了解其他区域的情况

### 2.6 玉衡山（江南·妖域）

| 项目 | 内容 |
|------|------|
| **所属区域** | 江南·深山 |
| **定位** | 妖怪聚集的山脉，妖界社交场 |

| **经济** | 妖界经济——月酒、灵果、妖物材料 |

- 深处有凶妖——它们是被污染灵气催生的恶妖（与外面的善妖形成对比）

## 三、中原（中后期区域）

> **总氛围**：曾经的天下中心，如今断壁残垣。朝廷苟延残喘，国师是最后的支柱。秦宗权残部在南部肆虐，各路势力在此交汇。这里没有"太平"——只有废墟上的权力游戏。但正统佛道的根基还在，丹药的残存产地也在这里。

### 3.1 残存朝廷所在地

| 项目 | 内容 |
|------|------|
| **所属区域** | 中原·某处（朝廷流亡地） |
| **定位** | 朝廷最后的据点，中后期区域 |
| **氛围** | 破败的行宫，稀少的朝臣，疲惫的禁军；国师是唯一还有实力的人 |
| **势力** | 残存朝廷、国师、镇抚司残部 |
| **妖鬼** | 极少（国师的法力镇压）——但暗中有异 |
| **NPC** | 镜抚司残部、朝廷官员（大多是废物）；国师已去世，但有人记得她 |
| **经济** | 朝廷最后的丹药储备在这里；但数量极少 |
| **功能** | 接触国师线、了解"太平篇"的前置信息、获取稀有丹药 |
| **落脚处** | 镇抚司据点 |

**设计要点**：
- 国师曾经是这里唯一有实力的人——但她已经去世了
- 朝廷在国师去世后更加衰败——连最后的支柱都没了
- 朝廷官员大多是尸位素餐的废物——皇帝是傀儡
- 镜抚司还有几个有能力的人在坚持
- 玩家在这里能感受到：**一个曾经伟大的体制已经死了**
- 从这里的NPC口中可以了解到国师的过去——"她活了快两百年""她最后回故乡去了""再也没回来"
- **第三章前置**：玩家根据燕大侠的建议来到此地，才得知国师已死，获得国师留下的信，线索指向她的故乡

---

### 3.2 洛阳废墟（中原·佛道旧址）

| 项目 | 内容 |
|------|------|
| **所属区域** | 中原 |
| **定位** | 曾经的佛道总坛所在，如今大半荒废 |
| **氛围** | 断壁残垣中还有几座道观/寺庙在坚持；修行者凋零但传承未断 |
| **势力** | 残存的正统道教/佛教、少数坚守的修行者 |
| **妖鬼** | 废墟中有大量游魂（战乱死者）；正统修行者勉强镇压 |
| **NPC** | 高级导师（可学高级术法）、正派僧道、钦天监残部 |
| **经济** | 丹药残存产地（极少量）、高级法器偶有流通 |
| **功能** | 拜师学高级术法、获取丹药、了解正统修行体系 |
| **落脚处** | 道观/寺庙（教派名声够高可借宿） |

**设计要点**：
- 与江南千佛寺形成对比——这里是"正统"，虽然衰败但底线还在
- 正统教派对千佛寺的态度：痛心疾首但无力管辖（自身难保）
- 玩家可以在此学到高级术法/符式
- 丹药在此地能获得少量（国师的炼丹传承还有残余）
- 这里的修行者知道更多关于"灵气污染"和"妖魔起源"的知识

### 3.3 秦宗权控制区（中原·南部/高危区域）

## 九、妖魔起源与分布逻辑

### 9.1 核心公式

> **怨气/杀业 → 灵气污染 → 妖魔滋生**

```
正常状态（盛唐时期）：
  天地灵气清净 → 仙家（土地/山神）维护秩序 → 妖魔极少

当前状态（唐末乱世）：
  人吃人、战乱、屠杀 → 大量怨气/杀业/死气 → 灵气被污染
  → 污染的灵气催生妖魔（尸体异变成僵尸、怨灵不散成鬼、动物吸收污染灵气加速成妖）
  → 妖魔越多 → 杀戮越多 → 怨气越重 → 灵气越污染 → 妖魔更多
  → 恶性循环
```

### 9.2 仙家为什么管不住？

| 原因 | 说明 |
|------|------|
| **灵气被污染** | 仙家的力量来源于清净灵气，灵气被污染后仙家也在衰弱 |
| **数量碾压** | 乱世制造的怨灵/妖魔数量远超仙家能处理的上限 |
| **香火断绝** | 土地公/山神的力量来自百姓供奉，百姓死光/逃光，仙家消散 |
| **造化丹加速** | （暗线）造化丹的炼制在大规模制造怨气和污染灵气 |

### 9.3 各区域妖魔密度

| 区域 | 妖魔密度 | 原因 |
|------|---------|------|
| **玉衡山** | 低（善妖为主） | 远离人间杀戮，灵气清净 |
| **吴越/闽地城内** | 极低 | 有秩序，有修行者镇压 |
| **江南野外** | 中等 | 有杀戮但不算最严重 |
| **千佛寺周边** | 高 | 大量炼制造化丹产生的怨气 |
| **中原** | 高 | 战乱反复，怨气深重 |
| **秦宗权控制区** | 极高 | 吃人军队制造的怨气最重 |
| **北境战场** | 极高 | 大规模战争的亡魂 |

## 十二、设计原则

1. **整个世界都是乱世**——没有"太平盛世"，只有"相对不那么乱"的地方
2. **有绝望也有希望**——钱镠、王审知代表"乱世中的光"；秦宗权、千佛寺代表"人间地狱"
3. **玩家自由选择节奏**——可以一直在江南接委托，也可以直奔主线
4. **妖魔密度与杀戮程度正相关**——越是血腥的地方，妖魔越多
5. **经济反映乱世**——物资匮乏、以物易物、丹药极度稀缺
6. **每个地点都有故事**——不是空壳地图，每个地点都有NPC、事件、秘密
7. **历史原型增加真实感**——钱镠、王审知、李克用、朱温、秦宗权都有原型，让世界更可信
8. **"黄角式"组织是希望的种子**——穷苦百姓的互助，白莲教的雏形，代表底层的自救
