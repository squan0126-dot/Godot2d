# 参考资料：《地煞七十二变》设定整理

> 原作者：祭酒 | 378章，约140万字 | 来源：网络公开书评、百科、章节目录、读者讨论  
> 用途：为项目世界观设计提供参考，不代表最终设计方向

---

## 一、核心设定

| 项目 | 内容 |
|------|------|
| 主角 | 李长安，自号玄霄 |
| 穿越触发 | 爷爷葬礼上翻开一本黄壳书，被送回古代世界 |
| 世界结构 | 现代世界 ↔ 唐末架空古代世界，双线穿梭 |
| 核心道具 | **黄壳书**——书页上画着妖魔，斩杀后获得神通 |
| 核心循环 | 去古代 → 斩杀妖魔 → 回现代获得神通 → 时限内必须再去古代（否则妖魔被召唤到现代） |
| 神通体系 | 地煞七十二般变化，每斩杀一个妖魔解锁一项（通幽、剑术、斩妖、御风等） |
| 世界观调性 | 唐末乱世：妖食人，鬼杀人，官匪抓人，人人皆苦，朝不保夕 |
| 叙事特色 | 文笔有《酉阳杂俎》《聊斋志异》韵味，融合古典志怪与现代灵异 |

---

## 二、世界观氛围

- 时代：唐末架空世界
- 官府腐败无能，叛军来势汹汹
- 妖魔横行，人心诡谲
- 主角是一把剑对抗无尽黑暗的侠客
- 氛围：**恐怖压抑 × 侠义豪情**，黑暗是底色，侠气是光

---

## 三、各篇章剧情概要

全书共6-7个主要篇章（副本），每个篇章在古代世界是一个独立的志怪故事，现代世界穿插推进长线悬念。

---

### 第一篇：画皮鬼篇（第1-6章）

**章节**：七十二变 → 人头面 → 老道 → 画皮 → 制服 → 鬼婴

**剧情概要**：
- 李长安在爷爷葬礼上翻开黄壳书，穿越到古代世界
- 遭遇"人头面"（一种偷人脸皮的鬼物），以普通人身份仓促应对
- 遇到一位**老道**，跟随其学习基础道术——符箓、超度、道教战斗法门
- 协助老道处理"画皮鬼"事件——一只披着人皮的恶鬼在村落中作祟
- 制服画皮鬼后，又接了一个"鬼婴"委托：富户家中死婴怨气不散
- 通过这两桩事，主角初步掌握驱鬼除妖的方法
- 画皮鬼斩杀后，黄壳书首次响应，获得第一项神通

### 第二篇：僵尸王篇 + 白狐篇（第7-22章）

**章节**：一意孤行 → 荒山破庙美人 → 斗 → 鬼屋 → 柳枝打诡 → 噩梦 → 符咒与剑术 → 鬼脸 → 夜黑勿开灯 → 狐狸 → 白狐拜月 → 突袭 → 电锯斗僵 → （20章缺失） → 斩妖 → 空壳

**剧情概要**：
- 老道离开/隐退后，主角独自上路
- 途经荒山破庙，遭遇幻化为美人的鬼魅，一番斗法
- 鬼屋除妖，用柳枝驱诡，习得符咒与剑术的配合
- 现代线穿插：夜黑勿开灯（现代世界的诡异事件）
- 白狐拜月：遇到一只修炼的白狐，善恶难辨，引出狐妖线
- 僵尸王出现，主角用电锯+道术与僵尸战斗（电锯斗僵）
- 最终斩杀僵尸王，获得第二项神通

### 第三篇：山蜘蛛篇（第23-47章）

**章节**：山宴 → 飞剑 → 飞剑？ → 飞飞 → 云浣纱 → 山雨见荒村 → 尸宴 → 尸潮 → 期望 → 灯会 → 梁上客 → 剑器舞浑脱 → 雾 → 仙山？ → 佳人有约 → 山蜘蛛 → 绝境 → 绝境如何逢生 → 饵 → 赶着投胎 → 天干物燥宜用火攻 → 虫巢 → 女警 → 御风 → 新受害者

**剧情概要**：
- 主角来到山间，参加了一场诡异的"山宴"——实为妖魔的陷阱
- 获得飞剑之术，尝试御剑，经历坎坷（飞剑？飞飞？）
- 遇到关键女性角色**云浣纱**——此后成为重要伙伴
- 山雨之中深入荒村，发现整个村庄已沦为尸宴现场——村民全被妖魔控制
- 尸潮爆发，主角与幸存者死守
- 经历灯会、梁上客等较小的妖异事件，世界观逐渐展开
- **山蜘蛛**本体出现——一只巨大的山蜘蛛精，控制着整个山脉的妖魔生态
- 主角陷入绝境，被山蜘蛛困于虫巢之中
- 用火攻烧毁虫巢，配合新领悟的御风之术，最终击毙山蜘蛛
- 山蜘蛛斩杀后获得"御风"等神通

### 第四篇：千佛寺/尸佛篇（第48-68章）

**章节**：白修业 → 蛇球 → 追击 → 问冤仇 → 魂灯 → 日记 → 风火 → 不愿作佛 → 雨行僧 → 僧鬼 → 肉香 → 斗 → 口信 → 村宴 → 京观 → 怨煞 → 讨魔校尉燕 → 忍死 → 鬼太守 → 豺狼 → 白莲降世

**剧情概要**：
- 主角进入**郁州**，当地有一座香火极盛的千佛寺
- 表面香火鼎盛，内里弥漫着诡异的氛围——夜晚寺庙飘出肉香
- 一位法号**圆通**的和尚（本是泼皮避兵灾做了和尚）是线人角色
- 千佛寺制作"肉身佛"：将活人做成佛像供奉，实为邪术
- 寺中僧众贪婪暴虐，以佛法之名行妖魔之事
- 寺庙深处供奉着三身佛，但三身佛早已**坠入魔道**，化为**尸佛**
- 寺庙下累积了无数尸骨（京观），怨气凝聚成煞
- 关键时刻引入"**讨魔校尉燕**"——朝廷派来处理妖魔的军官
- 同时出现**鬼太守**：已成为鬼的地方官
- **白莲教**（白莲降世）作为另一股势力开始浮现
- 高潮：与尸佛的正面对决——尸佛是全书目前最震撼的Boss战之一
- 宁死不屈的忠义老员外，白发苍苍提剑入阵——全书经典场景

### 第五篇：钱塘/孽龙篇（第69-101+章）

**章节**：镇龙锁 → 重围 → 神射 → 无遮大会 → 夜半来客 → 人殉 → 山君有请 → 雾谷 → 妖巢 → 虿鬼 → 人脍 → 重围 → 凶神恶煞 → 贪痴 → 葫芦书生 → 化魔窟 → 羊叩首 → 造畜 → 围捕 → 夜雨 → 追杀 → 废墟 → 怪雾 → 客栈 → 看破 → 图穷匕见 → 秤恶量善 → 往事与波折 → 决绝 → 幽冥

**剧情概要**：
- 主角来到**钱塘**，这是一个江河交汇的商业重镇
- 钱塘水下镇压着孽龙的"镇龙锁"——传说一旦被破，孽龙走水，全城覆没
- 钱塘的轮回体系由**轮转寺**掌管，但寺庙腐败无能
- 钱塘地界上的鬼魅越来越多，阴阳平衡正在被打破
- **无遮大会**：寺庙举办的法会，名义上是超度亡灵，实为敛财
- **人殉**：活人被献祭给山中的"山君"（某种强大的精怪）
- **造畜**：一种邪术，把人变成牲畜
- **虿鬼**、**人脍**：各种恐怖妖异
- 主角在一系列探索中逐步接近真相：孽龙正在苏醒
- 如果孽龙走水成功，钱塘将增加十余万新鬼，届时钱塘将五分人五分鬼，阴阳彻底失衡
- **葫芦书生**：一个神秘角色，可能掌握关键信息
- 关键对决：与孽龙相关的各方势力展开最终博弈
- 大量支线角色登场：船主、粮商、保义团等钱塘各方势力
- 结局牵动钱塘数十万人的生死

### 第六篇：潇水篇（约第38108行~第66000+行）

**章节**：浮舟说鬼 → 江神 → 鬼村 → 潇水 → 惊醒 → 凶案 → …… → 酒神祭 → …… → 记忆 → ……

**地点**：潇水——群山环绕、水网密布的酿酒小城

**核心悬念**：整座繁华的城市是一个幻境——所有的安居乐业都是假的

**剧情概要**：
- 李长安渡河时遭遇"江神"（水鬼），用青铜短剑镇压
- 进入潇水城——一座乱世中不可思议的繁华小城，满城紫藤萝，酒业兴盛
- 黄壳书上画的不是妖魔，而是一座城——暗示整座城市有问题
- 城中极度"干净"——闻不到任何妖气（反常）
- 夜间发生凶杀案——住客钱大志被杀，尸体上有妖气
- 发现"暴食"现象——多人突然食量暴增，然后被杀
- 遭遇**鬼面人虞眉**——红裙素衣、狰狞面具的女子，在夜间杀人
- 虞眉声称自己是镇抚司的人，杀的"从来不是人"——是即将变成妖怪的人
- **妖疫**：有人在散布能将人变作妖怪的疫病/咒术
- 夜间雾气弥漫时，"人"会变成传说中的怪物（虎姑婆、俎鬼等）
- **俎鬼事件**：周淮的父亲变成了食腐妖怪，周淮选择包庇父亲
- 酒神祭期间，全城雾气弥漫——大规模妖变即将爆发
- **真相揭露**：李长安用风火雷烧穿幻境，发现——
  - 整座潇水城是一个**幻境**，真实的潇水早已是废墟
  - 城中"居民"全是妖怪，被注入虚假记忆扮演人类
  - **始作俑者：俞梅**——闾山派上代掌教真人，年迈后回到故乡，用毕生法力和捕获的妖怪重建儿时记忆中的盛世潇水
  - 俞梅死后，**于枚**（紫藤萝精，幻境中化身"青萍真人"）违背遗愿继续维持幻境
  - 为维持幻境，于枚用大雾吞噬路过的真人——以血肉和魂灵为燃料
  - **虞眉**（大槐树精）不知道自己是妖怪，以为自己在"除妖"
  - **酒神**（潇水土地神）被封入石像，无力干涉——"君子不饮盗泉之水"
- 酒神向李长安展示"这片土地的记忆"——潇水从盛世到毁灭的全过程
- 最终Boss战：与于枚的对决

**核心主题**：
- 一个人用幻术重建已经毁灭的故乡——美好的代价是真实的死亡
- "表面的繁华太平" × "暗中的恐怖真相"
- 始作俑者不是恶人——只是一个想回到童年的垂死老人
- 维护者也不是纯恶——只是想活下去

**关键设定**：
- 幻境时间循环：酒神祭前后20天反复
- 妖怪扮演人类：被注入虚假记忆，精气不足时会失控恢复本性
- "怪"的概念：从人的恐惧/传说中诞生的妖魔（区别于"妖"）
- 五猖兵马：闾山派特有法术，用于维持幻境秩序

## 五、重要配角简表

- 侠肝义胆，义薄云天——不为变强而战，只为"该做的事"
- **"十步杀一妖，千里不留行"**——不求回报，只求问心无愧
- 始终站在弱者一边
- 不受门派宗门规矩约束，洒脱却有底线
- 从普通人到侠客的成长：初始什么都不会，跟随老道学习，一步步变强

## 六、可借鉴方向（供本项目参考）

| 方向 | 说明 |
|------|------|
| 斩杀妖魔解锁能力 | 核心成长循环，结合伴生道具 |
| 伴生道具的神秘感 | 黄壳书——不解释来源、不定义本质，留白制造悬念 |
| 开篇跟随导师修行 | 老道教基础（符箓、超度、道术），做小活儿铺垫，引出第一只大妖 |
| 志怪氛围 | 唐末乱世 + 妖鬼横行 + 官府腐败 + 民间疾苦 |
| 侠义精神 | 不为变强，只为"该做的事" |
| 妖魔各有来历 | 每个怪都不是凭空出现，背后都有人间的因果 |
| 世界观逐步展开 | 不在一开始就把设定全抛出来，让玩家/读者跟主角一起发现 |
| 多势力博弈 | 朝廷、教派、叛军、妖魔、民间——各种力量在乱世中相互交织 |
