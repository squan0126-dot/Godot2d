## 二、中原核心区（五代政权的舞台）

这是**五代政权的中心**——谁控制了这个区域，谁就能称帝。五代的五个朝代（后梁→后唐→后晋→后汉→后周）全部以这个区域为核心舞台。

### 2.1 三大都城

| 城市 | 地位 | 地貌 | 历史角色 |
|------|------|------|---------|
| **长安** | 唐都、旧日辉煌 | 关中平原，四面关隘（潼关、武关、散关、萧关） | 黄巢焚城后永久衰落；朱温迁都后沦为废墟；904年后退出历史舞台 |
| **洛阳** | 唐东都 → 唐末最后首都 | 伊洛盆地，四塞之地 | 904年朱温强迁昭宗至此；907年唐朝在此灭亡；五代时期地位下降 |
| **开封（汴州）** | 后梁首都 → 五代第一城 | 华北平原，汴河枢纽，无险可守 | 朱温建后梁定都于此；此后五代大多都开封；漕运决定了它的崛起 |

> 三城关系：长安代表"过去"、洛阳代表"过渡"、开封代表"未来"。从长安到开封的东移，是中国都城史的转折。

### 2.2 核心城市链

```
长安 —官道→ 潼关 —黄河→ 洛阳 —平原→ 开封（汴州）
                                          │
                                          ↓ 汴河
                                        宋州（今商丘）
                                          │
                                          ↓
                                        徐州
                                          │
                                          ↓
                                        扬州 → 长江 → 南方
```

这条线就是**黄河—汴河走廊**，是唐末五代的经济命脉。朱温控制汴州后，扼住了南北漕运，这是他崛起的关键。

### 2.3 中原的战斗地形

| 地形 | 位置 | 军事意义 |
|------|------|---------|
| 潼关 | 长安东大门 | 控制了这里，关中就是铁桶 |
| 函谷关 | 洛阳西面 | 洛阳的西屏障 |
| 虎牢关 | 洛阳东面 | 中原门户 |
| 濮州 | 山东，黄河边 | 王仙芝/黄巢起兵地 |
| 陈桥驿 | 开封北郊 | 赵匡胤兵变地（960年） |
| 柏乡 | 河北中部 | 李存勖大败后梁的决定性战场（911年） |
| 白马驿 | 滑州黄河边 | 朱温屠清流士族之地（905年） |

## 三、河东/太原——李克用的根据地

### 3.1 战略地位

太原是**北方最坚固的堡垒**。三面环山、一面临河（汾河），易守难攻。谁控制了太原，谁就控制了黄河以东的整个区域。

```
太原（河东节度使治所）
  ├── 北向：雁门 → 云州 → 进入草原（沙陀骑兵的来源地）
  ├── 东向：井陉 → 河北平原 → 幽州
  ├── 南向：潞州 → 泽州 → 洛阳/开封（直捅中原腹地）
  └── 西向：黄河 → 陕西 → 关陇
```

### 3.2 李克用的势力范围

| 城市 | 地位 |
|------|------|
| **太原** | 河东节度使治所，李克用的权力核心 |
| **潞州** | 太原与中原之间的关键节点，908年潞州之战 |
| **云州** | 北方门户，沙陀人南下的通道 |

> 梁晋争霸的地图本质：朱温从开封向北推，李克用从太原向南推，两个人在黄河以北反复拉锯。潞州（今山西长治）就是棋盘上反复易手的那个关键棋子。

> **一句话**：燕云十六州是中原的城墙。936年石敬瑭把这面墙拆了送人，此后186年中原活在契丹铁骑的阴影下。

### 4.3 对游戏的意义

燕云地区是"乱世行"的极端环境据点：
- **案五·燕云守碑人**就在此地——边境、烽火台、被遗忘的士兵
- 这里有独特的"边民文化"——汉人与契丹人混居，语言风俗交融
- 契丹"打草谷"——骑兵随意劫掠边境汉人村庄
- 大量汉人南逃，留下了无数荒村和废墟——**天然恐怖场景**

## 六、北境与塞外

### 6.1 契丹/辽的势力范围

| 区域 | 位置 | 特征 |
|------|------|------|
| **上京临潢府** | 今内蒙古巴林左旗 | 契丹首都 |
| **草原腹地** | 蒙古高原东部 | 契丹本部，骑兵来源 |
| **燕云十六州（936年后）** | 今北京、河北北部 | 汉人农耕区，契丹的"中原窗口" |

916年耶律阿保机建立契丹国后，北方出现了一个统一、强大的草原政权。这个政权不是传统意义上的游牧部落松散联盟——它是有文字、有制度、有南北面官体系的**帝国**。

> 五代中原的混战之所以无法统一，外部原因就是契丹的持续压力。北方藩镇要同时对付中原对手和契丹威胁，腹背受敌。

### 6.2 沙陀人的活动区域

沙陀是西突厥一支，原居中亚，唐朝中期内附，被安置在河东以北。

```
沙陀活动区域：
  云州（大同）以北 → 太原 → 上党（潞州）
  
李克用的沙陀骑兵（鸦军）是唐末最精锐的骑兵。
他们是唯一能在野战中击败黄巢义军的军事力量。
```

## 八、时间框架总览

游戏的核心时间段为 **875—960年**（黄巢起义到北宋建立），共85年。主角以"昏迷传送"方式在各时间点间跳跃，每次传送可能跨越数周到数年。

```
─────唐朝由盛转衰─────┬────唐末乱世────┬────────五代十国───────→
755          874    875          907   907                    960
│                   │                   │
安史之乱             黄巢起义            北宋建立
（转折点）           （开篇）            （终点——赵匡胤投胎后）
```

## 九、三段时期的分界

### 9.1 第一段：唐朝由盛转衰（755—874年）

**这段是背景，不是游戏。** 主角不会穿越到这个时期，但它是理解整个世界的必要前提。

| 关键节点 | 年份 | 对世界的影响 |
|---------|------|------------|
| 安史之乱 | 755—763 | 藩镇割据、宦官专权、经济重心南移——三个根本问题全部由此开始 |
| 甘露之变 | 835 | 宦官专权到达顶峰，皇帝彻底沦为傀儡 |
| 回鹘崩溃 | 840 | 沙陀人南下，李克用势力的起源 |
| 庞勋兵变 | 868 | 黄巢起义的"彩排" |

> 这段时期的核心逻辑：**唐朝不是突然灭亡的，是被安史之乱后遗症慢慢耗死的。**

### 9.2 第二段：唐末乱世（875—907年）

**这是游戏的主要时间窗口。** 主角的大部分案件发生在这一时期（或其刚结束的十国初期）。

| 年份 | 大事件 | 游戏意义 |
|------|--------|---------|
| **875** | 黄巢起义爆发 | 天下大乱的起点，妖鬼横行的社会根源 |
| **880** | 黄巢入长安 | 长安沦为废墟——废墟中留存了无数亡灵与执念 |
| **882** | 朱温降唐 | 朱温崛起，北方进入军阀混战 |
| **883** | 李克用勤王 | 沙陀势力正式登上中原舞台 |
| **884** | 黄巢死 | 起义结束，但唐朝彻底失去控制力 |
| **893** | 三强鼎立 | 朱温（中原）、李克用（河东）、李茂贞（凤翔） |
| **900—904** | 朱温控制朝廷 | 宦官被诛、昭宗被弑、士族被屠 |
| **907** | 唐朝灭亡 | 五代十国正式开始 |

> **开篇案的背景时间**：案一（汴州·饿妖）设定在907年深秋——唐朝刚亡、后梁初建的节点。黄巢败亡后的溃兵流落荒野，在饥饿中从人变成妖。主角在汴州郊外遇云游道士玄义，因能见鬼被留为帮手，后玄义老道于饿妖一战中力竭牺牲，主角继承其道袍、剑与瘦马，从此以道士身份独行乱世。吃人的兵将——这是唐末乱世结出的最毒的一颗果子；衣钵传承——这是乱世中最暖的一盏灯。

### 9.3 第三段：五代十国（907—960年）

**这是游戏的核心时间段。** 主角全部案件分布于此。

| 时间段 | 中原政权 | 主角案件参考 |
|--------|---------|------------|
| 907—912 | 后梁·朱温 | 案一汴州、案二洛阳 |
| 912—923 | 后梁内乱 → 灭于后唐 | 案三钱塘、案五燕云 |
| 923—936 | 后唐（李存勖→李嗣源） | （可插入案件） |
| 936—947 | 后晋（石敬瑭→契丹灭晋） | 燕云十六州割让，边境剧变 |
| 947—950 | 后汉（最短命，仅3年） | （可插入南方案件） |
| 951—960 | 后周（郭威→柴荣） | 柴荣治世——乱世中最好的时光 |
| **960** | **北宋建立** | 赵匡胤投胎的时间点 |

### 时间跳跃规律

| 跳跃类型 | 时间跨度 | 叙事效果 |
|---------|---------|---------|
| 短跳 | 数周—数月 | 连续感强，前案的余波还在 |
| 中跳 | 半年—一年 | 适度的疏离，前案已成为回忆 |
| 长跳 | 1—3年 | 历史背景明显变化，主角面临更大的陌生感 |
| 特殊跳 | 不定 | 穿插在关键历史节点（如燕云割让936年），制造历史震撼 |

## 十三、传送机制与时空叙事

### 13.1 主角的时间感知

```
现实时间线：
  穿越前（现代） → 案1 → 案2 → 案3 → …… → 案N → 玉佩亮满 → 赵匡胤投胎（960年？）

主角感知：
  他醒来 → 他不知道现在是哪一年
     ↓
  他从当地人口中得知"今上是谁"
     ↓
  他才明白自己又跳了几年
     ↓
  他掏出那部没信号的手机——
  日历停在2024年5月17日。
  再也没翻过一页。
```

### 13.2 不动点 vs 跳动点

| 类型 | 说明 |
|------|------|
| **不动点** | 主角自身（记忆、能力、物品）不随时间改变 |
| **跳动点** | 外部世界在跳（朝代更替、城市变迁、旧人老去） |
| **锚点** | 玉佩——唯一的时间坐标（刻痕数 = 进度） |

> 叙事张力源于：主角是时间中的漂泊者。他认识的每个人都会老去或被战乱吞噬，而他不断被抛向下一个陌生的时间点。他没办法"回去看看那个村子后来怎样了"——这就是"昏迷传送"的核心悲剧性。

### 13.3 赵匡胤魂魄的时间线

```
960年  赵匡胤建立北宋（正常历史）
  ↓
???   赵匡胤死后，魂魄被困入玉佩（原因成谜）
  ↓
约1000年间  玉佩辗转多人之手，多位"守玉人"接力
  ↓
2024年  玉佩出现在李归舟的水杯旁
  ↓
2024年→（古代）  李归舟开始在唐末乱世斩杀妖鬼
  ↓
每次斩妖 → 刻痕亮一道 → 赵匡胤离投胎近一步
  ↓
最终  刻痕全亮 → 赵匡胤在960年投胎 → 
      历史上的赵匡胤出生 → 完成闭环
```

> 这是一个**时间悖论式的闭环**：赵匡胤被困在玉佩里，需要未来的李归舟在古代斩妖积攒灵力，才能让赵匡胤在960年正常投胎出生——所以大宋之所以存在，恰恰是因为主角的穿越。
>
> 这意味着主角不是在"旁观历史"，他本身就是历史的一部分。没有他，就没有赵匡胤，就没有大宋。

## 十四、设计建议：案件时空分配

### 14.1 建议原则

1. **前5案集中在中原**（洛阳、汴州、太原、钱塘、燕云），让玩家先建立空间认知
2. **中期向南方扩散**（金陵、成都、江陵），展示南方九国的不同风味
3. **穿插关键历史节点案件**（如936年燕云割让、959年柴荣之死），用大事件增强历史感
4. **末期回到中原**（960年陈桥兵变前后），收束主线
5. **时间跳跃不宜太大**（避免跳跃超过3年），否则玩家失去连续感
